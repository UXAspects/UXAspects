To include a Back Button within the Standard Header, add the .back-button class to the nav element.

Within the section of the Standard Header which includes the “Breadcrumb and Heading” (see above), include a div with the .btn-back-header class.

Inside this, use the back-button-header directive to insert the Back Button. The target-state attribute should be used to set the target page for the button. It is recommended that this button should link to the previous level in the Breadcrumb trail.

The div containing the Breadcrumb and other Heading items should be enclosed in a div with the .header-content-left class.

For applications wishing to reuse the same Standard Header on pages both with and without a Back Button, toggling the .back-button class on the container will hide the Back Button.


To hide the Back Button on certain pages, introduce a global variable (inject as a global service or use $rootScope). Show the element contained in the btn-back-header class depending on the value of this variable.
