To enable programmatic selection of facets, pass an empty object to the api attribute of the facet-container element. This object will be populated with a list of objects representing the facets in the container, allowing access to the select and deselect methods of their facet-options.

This is compatible with standard Facets, Dynamic Facets, and Custom Facets. Note that when using Custom Facets, the value of the model needs to be changed independently of these functions.

An example of the use of programmatic selection can be seen in the Filter Popover Controller in the Detail Row Header documentation.
