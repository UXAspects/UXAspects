A marquee modal consists of two sections. The first is a panel on the left side and the second is the main content area to the right.

Side Panel

A side panel can be added using a div.side-panel. It can have an icon or image which should be contained in a div.marquee-logo. Any text to be displayed should be contained within a div.marquee-info-panel. If there is header text use an h1.header. For regular text use a p.description. On very small displays the side panel may be hidden to provide sufficient room for the primary modal content.

Main Panel

The main panel can be added using a div.main-panel. It can have a header using a div.marquee-header with a child h1.marquee-title that will contain the header text. The primary content of the modal should be contained within a div.marquee-body. To add a footer to the modal add a div.marquee-footer. Add any modal buttons to this footer element. The height of the modal is set based on the height of the contents of the main panel.

The marquee modal can be initialized using a controller which internally uses the $modal service. The $modal service provides an open method with the following options:

Set to false to disable animation on new modal/backdrop. Path to the modal template. Controller for the modal instance Alternative to controller-as syntax. Name of property controller will be instantiated into. Indicates whether a dialog should close on 'Esc'. Will set the width of the modal. Available options are 'sm', 'md' or 'lg'.

The open method returns a modal instance with the following options:

Closes the modal, passing the result. Dismisses the modal, with a reason. A promise that is resolved when the modal is closed and rejected when it is dismissed. A promise that is resolved when a modal gets opened after downloading content's templates and resolving all variables.
