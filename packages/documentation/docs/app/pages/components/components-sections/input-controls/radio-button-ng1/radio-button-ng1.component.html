To create a radio button use the radiobutton directive with an ng-model to set and store the value.

The following attributes can be set to customize the behavior of the control.

Initially sets the value of the radio button and will be updated to contain the current value if a user toggles the radio button. If this value equals the value in ng-model then the radio button will appear selected. If a user clicks the radio button ng-model will be given the value specified by this attribute. If set to true the radio button will be displayed without a border and background. If set to false the radio button will not be updated when clicking on it, can be used if something else is updating the state of the radio button.

Use the name attribute to indicate which radio buttons are part of the same group.
