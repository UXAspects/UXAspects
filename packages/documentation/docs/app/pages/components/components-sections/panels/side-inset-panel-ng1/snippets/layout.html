The Side Inset layout is a page pattern which provides an animated side panel for presenting additional information. The side panel is hidden or shown using the toggle button. The side panel is inset in the page and not overlaid, so the width of the main content area will be reduced to make space for the side panel.

To use the Side Inset layout, structure your page as follows:

  • A wrapper div following the page heading, with the side-inset attribute directive set.
    • A div containing the main page content, which needs to have the main-content class.
    • A div containing the side panel, which needs to have the side-inset class.

The order of the child elements will determine the position of the side panel - if the side-inset div is declared in the HTML before the main-content, then the side panel will display on the left of the screen. To place the side panel on the right of the screen, declare it after the main-content in the HTML.

On the wrapper element where the side-inset directive is declared, the following additional properties can be configured:

Name Type Binding Description Optional
side-inset-width number literal A value between 0 and 100, this is the percentage width of the parent element the expanded side inset panel will occupy (Default: 20). true
side-inset-button-top string literal A string that will be set as the top CSS position property of the toggle button, allowing its vertical placement to be configured (Default: '0px'). true
