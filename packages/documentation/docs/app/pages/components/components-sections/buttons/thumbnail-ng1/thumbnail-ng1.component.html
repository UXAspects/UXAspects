Thumbnails are a useful way of showing a concise description or preview of something (such as a file). It allows the user to catch a glimpse of the whole file without actually downloading or opening it.

Thumbnails can be easily implemented with the use of our thumbnail directive. This allows you to specify a thumbnail for an element and dynamically update the visibility of the thumbnail. There are some attributes you can specify on the thumbnail directive to customize it to suit your needs. These include:

Specifies the url of the image to be used as the thumbnail. If no url is defined the hpe-document icon will appear instead. Specifies whether or not the thumbnail should be visible. Defines the width of the thumbnail and the directive element. Takes % and px values. Defines the height of the thumbnail. Takes % and px values.
