Notifications can be used to inform the user of occurring events or to alert the user of something that may require their attention. Notifications will appear in the top right corner of the browser window and can be dismissed by the user by clicking the close button.

Notifications are controlled by the notificationService which can be injected into any controller in the application.

The notificationService has several functions available to control notifications.

This function should be called to show a notification. It should be given an object containing properties to configure the appearance and content of the notification. It will return the notification element (if notifications are visible - see setNotificationVisibility), this can be used to dismiss this individual notification at a later point. This function can be called to dismiss a notification. It should be passed the notification element to dismiss. This function will return an array of all notifications shown since the browser was last refreshed. The objects in the array returned will contain the properties passed to the showNotification function along with an additional date property which will contain the time at which the notification was shown. If a more permanent history of notifications is required consider using local storage or a database. This function will dismiss any currently visible notifications. This function will determine whether notifications should be visibly displayed on screen when the showNotification function is called. If the showNotification function is called and the visibility is set to false, no notification will show, however a record of it will still be kept and can be accessed using the getNotifications function. This function expects a boolean value as an argument, a value of true will display notifications and a value of false will hide them. The default value is true. This function allows you to set the order notifications are displayed. The two possible options are append (default) and prepend. When set to append new notifications will appear below older notifications. When set to prepend new notifications will appear above older notifications.

The following options can be used to configure the appearance and content of the notification:

Sets the icon from the icon set to display within the notification. Sets the title displayed at the top of the notification. Sets the text displayed within the notification. Sets the text displayed beneath the main notification text. This is useful for adding a timestamp to the notification. Sets the duration (in milliseconds) the notification should appear before automatically dismissing. A value of 0 will show the notification until manually dismissed. Sets the background color of the notification.
