Coordinates in the Live chart. Defines the chart type and its layout. Values include 'lines', 'bar' and 'points'.

The following attributes should be defined within series

Defines the thickness of the line chart. If set to true displays the chart. Specifies the colors of the fill. For Line chart this produces area graphs. If more than one color is specified this will produce a gradient. If set to true fills the chart with colors in the fillColors property. A number between 0 and 1 defines the opacity of the fill. Default size of shadows in pixels. Set it to 0 to remove shadows.

Live Charts are defined by the flot directive. The dataset and options attribute define the live chart data and options respectively. Dataset and Options are defined within the LiveChartCtrl.

Markings used to draw simple lines and rectangular areas in the background of the plot. It can be an array of ranges in the form of { xaxis: { from, to }, yaxis: { from, to } } or a function(above example) that returns this range.
