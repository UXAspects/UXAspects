To use this AngularJS component in an Angular application, import HybridModule into the module where it will be used.

The component name remains the same; however, the following should be considered when using this component in an Angular template:

  • Property names use camel case, so a property named on-next becomes onNext.
  • Angular binding syntax must be used, so a property with variable or function binding named valid would be written as [valid] for one-way binding, or [(valid)] for two-way binding (if supported).
