UX Aspects is the only open source user interface framework for building modern, responsive, mobile big data applications on the web. It provides all of the essentials required for your big data project, allowing you to swiftly create dashboards and reports that provide users with the insights they crave from their big data applications. UX Aspects is based on best-of-breed, open source technologies, including Angular, Bootstrap and D3. It is easy to include in your web applications with the Bower web package manager.

UX Aspects can be downloaded via the Bower package manager, allowing it to be added to your build process and work with your existing tools. If you donâ€™t need an automated install process, you can instead directly download the zip files. For installation details, refer to Getting Started.

UX Aspects provides you with the following key pieces for your user interface design:

  • A wide range of tables complete with sorting and filtering, card views to view data more easily on smaller screens and hotkeys to let you move through the site quickly and improve productivity.

  • Numerous charts to display your information in the best way helping you to make the most of your data. You can select from multiple line charts, bar charts, donut charts and many others. The charts can be stacked and configured to show forecast data.

Refer to Components, CSS and Charts for a detailed view of all the components or check out the Showcase for an example application created using UX Aspects.
