Tabs can be added under the header within a div with the navbar-tabs classes enabling the page content to be positioned in the tab content and the tabs to function as part of the header.

A toolbar must appear inside a div.affix-container.toolbarand can be placed inside the tab content.

Create a div.affix-toolbarelement which will contain all items in the toolbar. Add the affix-elementdirective to allow the toolbar to stick to the top of the page. In this example the affix-positionattribute has been added to affix the toolbar just below the condensed header.

The navigation bar can be presented in various different ways. By default the navigation bar appears in the standard mode with no divider.

To change the navigation bar to use the integrated appearance add the integrated-tabs class to the div.navbar-tabs element. To add a divider to the navigation bar add the nav-divider class to the div.navbar-tabs element.

A menu-tabcan be added to the tab-bar by adding the menu-tabclass to the tabset. The menu-tab directive is then added before defining the tabs.

The dropdown content is defined using the tab-menu-contentdirective which will require the following properties:

String for the title of the section. Array of strings containing the tab names.

The tabs are then created as normal with the headings matching the content from tab-menu-content and the hiddenclass added to any tabs which will be displayed in the menu
