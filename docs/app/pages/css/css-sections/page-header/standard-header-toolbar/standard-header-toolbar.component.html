Standard Header in a page with toolbar is not latched. You should remove the affix-element directive from the nav.navbar.navbar-static-top element as well as remove the surrounding div.affix-container.toolbar element.

A Toolbar must appear inside a div.affix-container.toolbar alongside the nav.navbar.navbar-static-top element.

Create a div.affix-toolbar element which will contain all items on the toolbar. Add the affix-element directive to allow the toolbar to stick to top of the page.

Toolbar controls can be extended with other controls available in UX Aspects. These controls are contained in the div.affix-toolbar element.

“Filter by…” search box is contained in the filter-text class.

“Filter…” text box is contained in the expand-input directive, explained in Input Expand.

Both the Standard Header (when condensed) & Toolbar can be fixed by adding the affix-position attribute to the affix-element directive in the toolbar.

The value of this attribute defines the top position of the toolbar in reference to the header. Also, the affix-element directive is retained in the standard header.
