Help text for input

Block level help text for form controls. Specify help text for an input using a span with .help-block. Any .help-block should have its id attribute populated. The id value should be used in the aria-labelledby attribute on the input.


Advanced Help Text

Only use the information icon when it is necessary to provide a more detailed description about a particular field.

Include an aria-label attribute to provide the same information to users of assistive technologies.

Place an i.ux-icon.ux-icon-information inside a div.form-information which should appear just after the .form-label.

Set data-toggle=tooltip, data-placement=right and the title attribute to the desired help text. Ensure to include the tooltip attribute.
