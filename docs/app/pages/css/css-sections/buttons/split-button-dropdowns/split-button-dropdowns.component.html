Split Button dropdowns are available in any size. Button dropdowns use the .btn-group class to group the button and the dropdown icon. The menu is contained in the .dropdown-menu class.

Split buttons are rendered as two separate buttons, one with the button text and the other drops down to open the menu.

Specify role="menu" on the .dropdown-menu element to better support assistive technologies. The aria-expanded="false" attribute should be specified on the data-toggle="dropdown" element as well.

Button dropdowns should use the aria-labelledby attribute so that assistive technology users know what the label for the menu is.

The element contained in the .sr-only class is used to improve experience with assistive technologies and is not rendered normally.
