The Primary Button is a button with the primary brand color, and should be used with the default action in a page.

Button colors can be used to prompt the user as to the type of action which will be performed; for example the .button-warning style can be used to warn a user about a potentially negative action.

The color of buttons will not be conveyed to users of assistive technologies such as screen readers.

If the button color is signifcant, a .sr-only tag should be added to describe this information. This label will be available to assistive technologies, but it will not be visible in the page.

Buttons are inline-block level elements so whitespace in the HTML of the page is significant and will affect their layout.
