ALERT! ux-icon-alert

Use the ux-icon class along with any class in the corresponding icon set listed above to create an icon.

The example above uses i tags but it can also be used with span tags.

If using an icon for decoration to avoid unintended or confusing outputs from assistive technologies you should use aria-hidden="true". If the icon has meaning, specify a label using sr-only to convey that information to assistive technologies.
