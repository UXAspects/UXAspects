Float Labels allow for placeholders that will move to remain visible when the user is typing in the input field. This pattern is useful for forms where having separate labels for inputs is undesirable, and allows the user to keep the context of an input field even after they have entered a value.

Float labels should not be used together with input addons, which already provide context to the input.

To apply a float label to an input, place the .float-label class on a div element. Place input and label elements inside; the label must be the first element after the input.

Set the placeholder attribute on the input to be the initial text that should be shown as the standard placeholder, then set the text of the label to be what should be shown when the user is typing.

Depending on the purpose of the input, this text may either change to provide further context, or stay the same.

The input should have the data-value attribute set. The input should have the data-toggle attribute set to floatLabel.

For a larger or smaller input, use the .input-lg and .input-sm classes on the input as required.
