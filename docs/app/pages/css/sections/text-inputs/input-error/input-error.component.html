(error)

Ux Aspects includes a validation style for error. This can be achieved by adding .has-error along side .form-group. Place an icon inside a span.form-control-feedback and add .has-feedback to the .form-group to display it.

When conveying state it is important that they are also conveyed for users of assistive technologies. We can use aria-describedby attribute along with the .sr-only to describe the state.
