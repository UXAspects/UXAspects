Button dropdowns are available in any size. Button dropdowns use the .btn-group class to group the button and the dropdown icon.

The menu is contained in the .dropdown-menu class. Specify role="menu" on the .dropdown-menu element to better support assistive technologies.

The aria-expanded attribute should be specified with a value of "false" on the data-toggle="dropdown" element as well.

Button dropdowns should use the aria-labelledby attribute so that assistive technology users know what the label for the menu is.
