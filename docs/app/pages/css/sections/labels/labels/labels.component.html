Use span.label to create a label. UX Aspects offers a wide range of label & outline label colors. Set the color of the label using one of the following classes:

.label-accent .label-outline-accent
.label-alternate1 .label-outline-alternate1
.label-alternate2 .label-outline-alternate2

.label-vibrant1 .label-outline-vibrant1

.label-grey1 .label-outline-grey1

If the colour of the label is being used to convey meaning, include an aria-label attribute to describe this information to users of assistive technologies.
