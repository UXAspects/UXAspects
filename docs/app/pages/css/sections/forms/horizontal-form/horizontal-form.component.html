To make a Horizontal Form you must first add the .form-horizontal to the surrounding form.

Each control must also be wrapped in .form-group. Think of each .form-group as a new row in our form. For controls with labels we must add .control-label, .form-label and column width .col-sm-* classes. We also need to specify the width the control can take up so we wrap the control in a div and give it a column width .col-sm-*.
