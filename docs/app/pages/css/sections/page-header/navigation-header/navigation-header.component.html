The UX Aspects Logo is contained in the navigation header section of the navigation menu

Navigation header can be styled by adding nav-header-dark and nav-header-light class to the root element for dark and light themes respectively. Navigation header can be condensed by adding condensed-panel class to the body element upon scrolling.

While using assistive technologies, add attribute aria-hidden="true" to the logo image.

To dynamically bind a product logo here, the img element should be used in place of a, using the ng-src attribute to bind a property containing the image URL.
