To include a Content Panel in the Standard Header, add the .content-panel class to the nav element.

Within the section of the Standard Header which includes the “Breadcrumb and Heading” (see above), include a div with the .content-header-right class.

The contents of this div can be customized as required. This panel will be hidden when the viewport becomes too narrow, and will not appear on Condensed Headers.

The Header for this page includes an example of a Header Content Panel, in this case illustrating how the header space can be used to show analytics.
