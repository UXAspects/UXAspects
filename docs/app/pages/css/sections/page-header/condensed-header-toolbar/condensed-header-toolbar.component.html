Condensed Header is contained in a nav.navbar.navbar-static-top.condensed element and the condensed-header directive with no attributes and is contained in a div with affix-container class.

The affix-element and affix classes provide fixed condensed header.

Condensed Toolbar must appear inside a div.affix-container.toolbar along side the nav.navbar.navbar-static-top.condensed element.

Create a div.affix-toolbar element which will contain all items on the toolbar. Add the .condensed class to make the toolbar condensed. Add the affix-element directive to allow the toolbar to stick to top of the page.

“Filter by..” text box is contained in the expand-input directive, explained in Input Expand.

Both the Condensed Header & Toolbar can be fixed by adding the affix-position attribute to the affix-element directive in the toolbar. The value of this attribute defines the top position of the toolbar in reference to the header. Also, the affix and affix-element classes are retained in the condensed header to fix it to the top. The toolbar fixes beneath the condensed header.
