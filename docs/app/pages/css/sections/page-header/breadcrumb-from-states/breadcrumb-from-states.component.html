A breadcrumb can automatically be constructed by using the UI Router states. Each child state will appear next to its parent in the breadcrumb. When defining states using the $stateProvider specify a property called ncyBreadcrumb which can have the following properties:

Property Description
label This is the name that will appear for this state in the breadcrumb.
parent This can be used to overwrite the parent state. Specify the name of the state.
skip This can be used to skip the state in the breadcrumb.
