UX Aspects provides all the essential components required for creating a big data application, allowing you to swiftly create dashboards and reports that provide users with the insights they crave from their big data applications. UX Aspects is based on best-of-breed, open source technologies, including Angular, Bootstrap and D3. It is easy to include in your web applications with NPM.

Angular

UX Aspects offers a wide range of components developed with Angular, allowing you to add and configure components quickly and easily in order to create a high quality application to meet all your big data needs.

Analytics

UX Aspects provides a wide range of configurable and customizable charts, including bar charts, line charts, and donut charts offering multiple ways to visualize data with the option to show forecast data. UX Aspects also offers a number of more complex charts, such as:

  • The social chart for displaying social interactions within a network of people.

  • The partition map for visualizing the breakdown of files within a repository.

For detailed information and functional examples of these charts refer to Charts.

Accessible

UX Aspects Angular components have been reviewed for accessibility, featuring extensive keyboard and screenreader support. The color scheme has also been reviewed for adequate contrast ratio where applicable. Documentation examples show how to apply ARIA roles and attributes along with UX Aspects components to create application features which comply with the Web Content Accessibility Guidelines version 2.0 (AA level).

Responsive

UX Aspects components are designed to be responsive so your applications will work well on a wide range of devices from mobiles and tablets to desktops.
