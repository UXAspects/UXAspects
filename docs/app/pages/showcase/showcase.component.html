List View Examples

A sample application illustrating the use of list views for large amounts of data. Features a range of UX Aspects components:

  • Layout Switcher: Present data in table view or card view and auto-switch views based on the display size.
  • Toolbar: Quick access to the most common actions.
  • Facets & Filters: Narrow your search by date, author, file type etc.
  • Sorting: Sort data either by using a dropdown menu or by clicking on a table column header.
  • Selection: Select one or more items to enable actions to be applied.
  • Document Preview: Using the preview pane.
  • PDF Download: Export table data to a downloadable PDF.
