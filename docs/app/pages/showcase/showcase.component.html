List View Examples

A sample application illustrating the use of list views for large amounts of data. Features a range of UX Aspects components:

  • Layout Switcher: Present data in table view or card view and auto-switch views based on the display size.
  • Toolbar: Quick access to the most common actions.
  • Facets & Filters: Narrow your search by date, author, file type etc.
  • Sorting: Sort data either by using a dropdown menu or by clicking on a table column header.
  • Selection: Select one or more items to enable actions to be applied.
  • Document Preview: Using the preview pane.
  • PDF Download: Export table data to a downloadable PDF.

Social Chart Example

A sample application illustrating the use of the social chart for viewing connections between people. This provides the following features:

  • Clickable Nodes: Select a node to focus on a specific person and their connections.
  • Clickable Edges: Select an edge (connector line) to focus on connections between two people. Click on the background again to zoom out again.
  • Hoverable Nodes and Edges: Hover over a node or an edge to see a summary popup.
  • Zoom: Zoom in or out of the chart and drag with the mouse to control the chart view. Double-click on the background to zoom right out.

Partition Map Example

A sample application illustrating the use of the partition map to show the breakdown of a document set by various criteria. This provides the following features:

  • Document breakdown: See a visual representation of the document set by custodian, language and file type.
  • Clickable Segments: Select a segment to zoom in or out of the partition map.
  • Editing: Edit the data types and their row order in the partition map.

Sankey Chart Example

A sample application illustrating the use of the sankey chart to show document flows within a system. This provides the following features:

  • Sections: View the sections of the system at each stage in the document flow.
  • Flows: See a visual representation of the document flow by hovering over a segment or the flow connections between adjacent segments.
