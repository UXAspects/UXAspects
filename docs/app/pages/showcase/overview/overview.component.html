Partition Map Example

A sample application illustrating the use of the partition map to show the breakdown of a document set by various criteria. This provides the following features:

  • Document breakdown: See a visual representation of the document set by custodian, language and file type.
  • Clickable Segments: Select a segment to zoom in or out of the partition map.
  • Editing: Edit the data types and their row order in the partition map.

Organization Chart Example

A sample application illustrating the use of the organization chart and hierarchy bar to show a hierarchy of different individuals within an organization. This provides the following features:

  • Clickable Nodes: Select employee nodes to navigate down through the hierarchical structure
  • Searching: Search through the organization for a particular employee.
  • Employee breakdown: See a visual representation of the employee including name, job title and email address.

Sankey Chart Example

A sample application illustrating the use of the sankey chart to show document flows within a system. This provides the following features:

  • Sections: View the sections of the system at each stage in the document flow.
  • Flows: See a visual representation of the document flow by hovering over a segment or the flow connections between adjacent segments.
