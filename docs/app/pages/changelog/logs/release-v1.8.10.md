UX Aspects 1.8.10 is available! Check out the documentation at [uxaspects.github.io/UXAspects](https://uxaspects.github.io/UXAspects).

#### Bug Fixes and Improvements
* (EL-3705) [Date Range Picker](https://uxaspects.github.io/UXAspects/#/components/date-time-picker#date-range-picker) - start/end labels for date and time can now be localized using `dateFormat` and `timeFormat`.
* (EL-3706) [Toolbar Search](https://uxaspects.github.io/UXAspects/#/components/search#toolbar-search) - adding default padding and the `ux-toolbar-search-clear-offset` class to prevent input text from overlaying the button(s).
