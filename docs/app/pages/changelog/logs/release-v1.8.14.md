UX Aspects 1.8.14 is available! Check out the documentation at [uxaspects.github.io/UXAspects](https://uxaspects.github.io/UXAspects).

#### Bug Fixes and Improvements
* (EL-3660) [Typeahead](https://uxaspects.github.io/UXAspects/#/components/input-controls#typeahead) & [Select](https://uxaspects.github.io/UXAspects/#/components/select#select) - added `auto` option to `dropDirection` which causes the dropdown to grow upwards when the component is close to the bottom of the viewport.
* (EL-3776) [Number Picker](https://uxaspects.github.io/UXAspects/#/components/input-controls#number-picker) - updating the max value now triggers an update to the validation state.
* (EL-3777) [Tabs](https://uxaspects.github.io/UXAspects/#/components/tabs#tabs) - added `selectTab` function which can be used to programmatically select a tab via index or instance.
* (EL-3779) [Side Panel](https://uxaspects.github.io/UXAspects/#/components/panels#side-panel) - added `closeOnEscape`, allowing the default behavior of closing the panel when escape is pressed to be disabled.
* (EL-3790) [Marquee Wizard](https://uxaspects.github.io/UXAspects/#/components/wizard#marquee-wizard) - fixed an error which occurred when loading steps asynchronously.
* (EL-3792) [Icon Set](https://uxaspects.github.io/UXAspects/#/css/icons#ux-icons) - added `platform-kubernetes`.
