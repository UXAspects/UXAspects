UX Aspects 5.1.0 is now available! Check out the documentation at [uxaspects.github.io/UXAspects](https://uxaspects.github.io/UXAspects).

#### Installation
```bash
npm install @ux-aspects/ux-aspects@5.1.0
```

#### New Features
* (EL-4282) [Marquee Wizard](https://uxaspects.github.io/UXAspects/#/components/wizard#marquee-wizard) -
Added a X icon beside steps which have a validation error.

### Documentation
* (EL-4282) [Select](https://uxaspects.github.io/UXAspects/#/components/select#select) -
Added section for validation in the documentation detailing how to add validation through the Reactive Forms API.
* (EL-4296) [Partition Map Example](https://uxaspects.github.io/UXAspects/#/showcase/visualizations/partition-map) -
Browser tab no longer hangs when using the example.
