As part of UX Aspects we provide two common prebuilt Facet list components, however you may require different facet components specifically for your application. We provide some useful classes for you to use to make the process of creating your own facet component simple while ensuring that it will work well with Facet Containers automatically.

Facet Header

First we recommend each facet component has a header, to identify what kind of type of filtering it will provide. It also allows a user to easily collapse and expand the component. We provide a header component for you to use as part of the FacetsModule. It can be added using the ux-facet-header selector.

The Facet Header component can use the following attributes to customize the behavior and appearance:

Defines the text to display in the header. Defines whether or not clicking on the header will toggle the expanded state. Can be used to set the initial expanded. If two-way binding is used it will be updated when the expanded state changes.

Creating a Custom Facet Component

Your custom facet component should extend the FacetBaseComponent component class. This base class will provide you with the following functions:

Will select the facet and update the selected values in the facet container. Will deselect the facet and update the selected values in the facet container. Will deselect all facets associated with this component and update the selected values in the facet container. If the facet passed to this function is currently selected it will be deselected. If the facet is currently not selected it will be selected. This will update the selected values in the facet container. Will return a true if the facet passed to the function is currently selected, otherwise will return false.

The base class also adds a selected two way binding and an events output which will automatically be triggered when facets are selected or deselected. The events property is an observable which can be subscribed to to watch for external events such as deselecting being triggered by the Facet Container.
