Custom Facets allow for any control, such as a Date Picker, to be used as part of a faceted search.

To include a Custom Facet in a facet-container, use the facet-custom directive in place of a facet directive. The required attributes are:

Name Type Binding Description Optional
name string variable A string name for the facet. false
model object variable An object in the controller context which contains the computed value of the custom facet. false
select function variable A function called when the value of model changes. false
deselect function variable A function called when the facet is deselected. false
display function variable A function which computes a display name for the facet when selected. false

The select and deselect functions are not called with parameters, so should be defined from the parent context to read either the value of the property passed as model or another property as appropriate.

Inside this directive, define the required control. The key feature is to ensure the computed value of the control is set to the same property as was passed as the model attribute for the directive. This means the custom facet is single select, and should implement its own clearing functionality if required.
