The Timeline component will display a vertical list of events. It will be defined by a timeline tag. Each event will be defined by a timeline-event tag.

For each event a badge and panel will be displayed. Information to be displayed in the badge and panel will be supplied as inputs to the Timeline event component.

The following attributes can be used to customise each event:

The color of the badge from the Color Palette. The badgeâ€™s text. If the width of the badge's text is greater than the width of its container, the text will be truncated to end with an ellipsis and a tooltip containing the text will be displayed upon hover. The event title. Displayed in the event panel. When the event occurred. Displayed in the event panel. Details of the event in HTML format. Displayed in the event panel.
