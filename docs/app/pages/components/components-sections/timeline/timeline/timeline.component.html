The ux-timeline component is a responsive, data-driven vertical component to tell a story, show history or describe a sequence of events. Events are created with the ux-timeline-event component, which includes a customizable badge and informational panel.

The main content of the event will be displayed in the informational panel and should be specified inside the ux-timeline-event element.

The following attributes can be used to configure the ux-timeline-event component:

The color of the badge from the Color Palette. A short piece of text to show in the badge; typically the short-form date of the event. The event title. Displayed in the event panel. When the event occurred. Displayed in the event panel.
