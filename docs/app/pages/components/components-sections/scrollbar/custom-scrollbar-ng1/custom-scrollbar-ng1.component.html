Custom scrollbar implements vertical scrollbar and does not support horizontal scrollbar.

Custom scrollbar can be used on elements with a relevant overflow property. The scrollbar can be added to an element by providing the scroll-pane attribute. The element should have an id or an attribute scroll-name which is assigned an id for the scrollbar to work. The options for scrollbar should be passed to the scroll-config attribute. Some of the options are:

Name Type Description
showOnlyOnHover boolean The scrollbar will be displayed only on hover when this option is set to true.
resizeSensor boolean The scrollbar will be re-initialised automatically when the content changes size or the viewport size changes. Use resizeSensorDelay property to add a delay before reinitialising.
autoReinitialise boolean An alternative to the resizeSensor property. This is a less performant option and uses polling to check for size changes, however in certain circumstances due to positioning of elements resizeSensor may not be suitable. The scrollbar will be re-initialised automatically when there is a change in size of the viewport.
enableKeyboardNavigation boolean The content will scroll using keyboard controls when it is given focus.
isScrollableH boolean Disables the horizontal scrollbar when this option is set to false.
scrollMargin number Margin in pixels to add around the scrollable area. The recommended margin as seen in the example above is 5. Default is 0.
contentUpdateSensor boolean This should be enabled if you have dropdowns within the scroll pane to allow the panel to resize when the dropdown is opened. It will be less performant with this enabled so should not be used unless needed.
contentUpdateSensorDelay number This will set the delay for the content update sensor. Lower numbers will have a greater impact to performance. By default this is set to 500ms.
