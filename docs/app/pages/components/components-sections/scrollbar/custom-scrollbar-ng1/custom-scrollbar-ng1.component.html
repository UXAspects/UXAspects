Custom scrollbar implements vertical scrollbar and does not support horizontal scrollbar.

Custom scrollbar can be used on elements with a relevant overflow property. The scrollbar can be added to an element by providing the scroll-pane attribute. The element should have an id or an attribute scroll-name which is assigned an id for the scrollbar to work. The options for scrollbar should be passed to the scroll-config attribute.
