The component-list directive allows you to add or remove instances of the same component. It should be given an ng-model containing an array. If the array has any items components will initially be created for those values. This array will be updated when a new field is added or when a component changes its model.

You should provide the html for the component you want to duplicate in a component tag inside the component-list directive. Each component will have a property called model available in its scope. Your component should update this property which will update the ng-model on the component-list.

Use the following attributes to customize the behavior of the component:

Specifies the minimum number of components that will be displayed. If no values have been set for these components in the ng-model then the default value they will be given is null. Specifies the maximum number of components that will be displayed. The add button will be disabled when the limit has been reached. By default there is no limit. Specifies the text to be displayed beside the add button. If specified the function will be called when a new item is being added to the list. If specified the function will be called when an item is being removed from the list.
