The different options to sort are contained in the sort-direction-toggle directive. The attribute label takes a string which is displayed before the sorter dropdown and the attribute descend is an optional attribute which can be set to true to default to sort by descending.

The attribute sorters stores the values required for sorting and should have the following attributes:

Name Type Binding Description Optional
name string variable String displayed in the dropdown list. false
sort string variable String indicating which sort will be applied. false
defaultSorter boolean variable Boolean value set to true to indicate which sort will be applied when the page loads, if none is specified the first object in sorters will be used. true
