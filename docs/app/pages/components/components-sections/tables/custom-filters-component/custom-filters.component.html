As part of UX Aspects we provide two common prebuilt Filter components, however you may require different filter components specifically for your application. We provide some useful classes for you to use to make the process of creating your own filter component simple while ensuring that it will work well with Filter Containers automatically.

Creating a Custom Filter Component

Your custom filter component should inject the FilterService. This service will provide you with the following functions:

Will add a filter to the list of active filters. Will remove a filter to the list of active filters. Will remove all filters from the list of active filters.
