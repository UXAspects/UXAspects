The different options to sort are contained in the sorter directive. The attribute sorter-title takes a string which is displayed before the sorter dropdown.

Each option to sort is specified using sorter-option directive.

sorter-option should have the following attributes:

Sorter option name displayed. Called when the option is selected. This method performs sorting, pagination and updates the data count. Allows a sorter option to be disabled. True if this is the default sorter. Class name of an icon to display on the right side of the sorter name (Example value: hpe-icon hpe-actions).
