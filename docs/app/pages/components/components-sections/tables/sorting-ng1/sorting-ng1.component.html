The different options to sort are contained in the sorter directive. The attribute sorter-title takes a string which is displayed before the sorter dropdown.

Each option to sort is specified using sorter-option directive.

sorter-option should have the following attributes:

Name Type Binding Description Optional
name string variable Sorter option name displayed. false
select function variable Called when the option is selected. This method performs sorting, pagination and updates the data count. false
disabled boolean variable Allows a sorter option to be disabled. By default sorter options are enabled. true
default boolean variable True if this is the default sorter. false
icon-class string variable Class name of an icon to display on the right side of the sorter name (Example value: hpe-icon hpe-actions). true
