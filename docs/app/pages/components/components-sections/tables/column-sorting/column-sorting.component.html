The component will be exported as ux-column-sorting.

Defines a unique identifier for the column that can be used when sorting. Defines the sorting order of a column: NoSort, Ascending or Descending. Changes the state of the sorting on the column between NoSort, Ascending and Descending. This returns an array of objects for each column being sorted containing key: string and state: ColumnSortingState. State can be used to find the current sorting state of the column eg. (state === ColumnSortingState.Ascending). The ColumnSortingOrder interface has been provided for objects in the array. Emits when state changes.

If set to true the column will sort by only this column, removing sorting from all others.
