These are actions that can be performed against a single item in a table. There should never be more than 3 of these actions and they should be restricted to the most common actions a user would want to perform.

To create these actions add the .item-actions class to the last td in each tr. Inside the td.item-actions add an item-hover-actions element which will contain the item-hover-action elements. The item-hover-action can have the following attributes:

A literal which is the name of the action and is placed in a tooltip. A literal for the icon class which is added to the action element along with .hpe-icon. A function which is executed when the action is clicked.
