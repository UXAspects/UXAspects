The dynamic showing and hiding of table columns can easily be achieved by using built in Angular directives.

Create an object in your controller containing a boolean property for each column in the table. These properties should be given an initial value that reflects whether or not the column should be visible.

Then we can simply add an ng-if directive to the table header cell and the table body cell pointing to the property matching that column in the controller.

The header and body cells will now automatically show or hide based on the value in the controller.

Note: Dynamic columns do not currently work with multiple sortable table headers
