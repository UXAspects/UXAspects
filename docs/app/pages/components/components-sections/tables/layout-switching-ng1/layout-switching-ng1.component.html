When using tables or cards to display information it may be desirable to switch between the different layouts either when toggled or automatically when the parent element begins to get too small. To implement layout switching use the layout-switcher-container directive. The directive must have the following attribute:

A bindable property which will have the name of the layout to display. A bindable property which, if specified, will store the layout name of the active layout. This differs from the selected value as if the layout is collapsed this property will contain the name of the collapsed layout.

Within the layout-switcher-container directive, layouts should be specified using the layout-switcher-item directive. They can have the following attributes:

The name of the layout which will be used by layout-switcher-container to indicate which layout to display. The URL of the HTML file to be used as the layout. If no layout is specified you should place the HTML of the template within the layout-switcher-item element. The name of the layout to be used when the display becomes small. The size at which the layout should change to the collapse layout.
