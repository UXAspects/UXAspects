Detail rows spanning columns can be added to responsive tables by adding divs with reponsive classes inside the table row. Add table and table-hover classes to the table element.

The first div represents the tabulated columns and their data. The next div represents the detail row that spans mulitple columns but is considered within the same table row.

Add any of the col-xs-*, col-sm-*, col-md-* col-lg-* classes to the inner div element to achieve responsiveness.
