Preview Pane is contained in preview-pane directive. This directive contains the following attributes:

Name Type Binding Description Optional
preview-file string variable A binding which specifies the file to be previewed. false
preview-title string variable An optional binding that when set will display a title. true
preview-subtitle string variable An optional binding that when set will display a subtitle. true
preview-empty-text string variable An optional binding that when set will display a text when preview pane is empty. true
shadow boolean literak An optional binding that will display a curved shadow on the bottom of the title or subtitle. true

Preview Pane is triggered by a Preview button defined in the preview-pane-toggle directive. The attributes of this directive are:

Name Type Binding Description Optional
set-preview boolean variable Defines if a Preview is enabled or disabled. false
tooltip string literal The tooltip for the toggle button. false
aria-label string literal Labels the preview button for assistive technologies. false

.hpe-divide-right or hpe-checkbox classes are used along with .hpe-lg class for larger size of icons, that shows if Preview is enabled or disabled.

Every item in the preview list is contained in the preview-pane-item directive. The directive defines the method that is executed when the row is clicked for preview. The user can implement the keydown-select attribute of this directive, set to true if he or she wishes to automatically select (click) a row when traversing a list with keyboard controls.

Define the tabIndex on each row to enable keyboard navigation.
