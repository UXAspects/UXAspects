The side modal view is defined by the side-modal-window directive. The input value of the directive is an object. It is defined by the vm.sideModalOptions object which in turn defines the modal options in the example provided.

The following defining the modal are set in the vm.sideModalOptions.

Name Description
title Defines the Header title of the modal view.
main Provides a path to a template representing modal content.
footer Provides a path to a template representing modal footer content.
modalColumns Classes defining the number of columns as per the grid system for a responsive modal.
affixHeader Boolean, defines whether header should be latched.
animate Boolean, defines whether the modal should open and close with an animation.

The modal can be closed or dismissed using the closeModal() and dismissModal() methods respectively.
