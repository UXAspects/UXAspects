The title of the Modal window is contained in the modal-header and modal-title classes. A dark modal header can be provided by using the modal-header-dark class along with the modal-header class and adding the modal-window-dark-header class as the value to the windowClass attribute of the modal.

The content and the buttons (footer) of the modal window are contained in the modal-body and modal-footer classes respectively.

The modal content height can be set in the modal-content class.

Modal View can be initialized using a controller which internally uses the $modal service.

The $modal service provides an open method with the following options:


Name Type Description Optional
animation boolean Set to false to disable animation on new modal/backdrop. If we don't specify the animation option by default animation is enabled true
template string Inline template representing modal content. true
templateUrl string Path to the modal template. true
controller string Controller for the modal instance true
controllerAs string Alternative to controller-as syntax. Name of property controller will be instantiated into. true
keyboard boolean Indicates whether a dialog should close on 'Esc'. Defaults to 'true'. true
size string Will set the width of the modal. Available options are 'sm', 'md' or 'lg'. Default is 'md'. true
Name Description
close(result) Closes the modal, passing the result.
dismiss(reason) Dismisses the modal, with a reason.
result A promise that is resolved when the modal is closed and rejected when it is dismissed.
opened A promise that is resolved when a modal gets opened after downloading content's templates and resolving all variables.

The OK and Cancel buttons in the modal view call these methods respectively.
