Add side-navigation directive to initalize the navigation menu.

Navigation menu is contained in the navbar-static-side-container class and can be styled by adding navbar-static-side-dark and navbar-static-side-light class to the root element for dark and light themes respectively.

The width of the navigation menu is set and can be modified in the navbar-static-side-container class.

If a parent navigation node does not have any content itself, the cascade class may be set on its child ul element. This will cause the click to activate the link associated with the first child item.

Note: In order to bold the text for each selected level, we use ng-class in our HTML to set the active class based on the page state. e.g. {{ noteCode }}
