An input mask can be useful when you want to restrict the format of the string a user can input. To create an input mask add a mask attribute with a valid format string.

The format string can take the following values:

  • 0 - digits (numbers)
  • 9 - digits (numbers), but optional
  • A - letters (uppercase or lowercase) and digits
  • S - only letters (uppercase or lowercase)
