The Number Picker provides a way in which users can easily select a number value from a specified range by clicking the up and down buttons or by using the up/down arrow keys on their keyboard.

To implement the Number Picker you must use the number-picker directive. There are some attributes you may specify on the number-picker directive to obtain the functionality you desire:

Describes the value the Number Picker is initialised with. It should be a value within the max and min range and must be declared. Describes the value the Number Picker is incremeted/decremented by each time. Describes the upper limit value. Describes the lower limit value. Describes the function that will validate the input value. It should return true if the input is invalid, false otherwise. Determines the disabled state of the Number Picker.

The Number Picker contains validation that will show if an input value is invalid which can occur for a number of reasons:

  1. The value doesn't fall between the specified range (max and min)
  2. The value is not a number
