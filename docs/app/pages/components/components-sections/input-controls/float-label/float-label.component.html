Float Labels allow for placeholders that will move to remain visible when the user is typing in the input field. This pattern is useful for forms where having separate labels for inputs is undesirable, and allows the user to keep the context of an input field even after they have entered a value.

Add the uxFloatLabel directive to the label element, and bind it to the associated input element.

The following properties can be used to configure uxFloatLabel:

The input element associated with the label. Whether the label is visible above the input field. This can be used to ensure that the label is positioned correctly when an input field is modified programmatically.

Determines which events cause the label to be raised.

  • focus: The label is raised on focus, and remains raised while the input contains text.
  • input: The label is raised when the user enters text. This mode is designed to be used in combination with the input placeholder attribute.
