Slider Chart Directive

The component can be added to a page by using the following HTML:

The slider chart has been primarily designed for use for filtering by date eg. within facets and should have the following attributes:

Property Type Binding Description Optional
ng-model Object Variable The range of values for the slider. false
slider-options Object Variable The options for the slider. false
chart-data Object Variable The data to populate the chart. false
chart-options Object Variable The options for the chart. false

Note: Refer to the Flot Options page for details of chart options.
