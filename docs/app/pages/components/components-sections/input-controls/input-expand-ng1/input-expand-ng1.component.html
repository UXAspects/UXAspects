Input Expand is contained in the expand-input directive and the .expand-input class.

Input Expand takes the following attributes:

Name Type Binding Description Optional
elname string literal Name of the input element. false
place-holder string literal Placeholder text for an input.
 false
class-name string literal Defines the classes for the input. false
focus function expression A method that defines a boolean value to indicate whether an input is expanded. Can be used to display or hide other elements. false
clear-text-icon string literal Icon used in the clear input text button. false
close-search string literal Should be set to a value which when clicked closes the input text false
expand-always boolean variable Indicates whether the search should always expand or only on a larger viewport. false
on-enter function variable Defines the function to be called when the enter key is pressed. The function will be called with the current text entered in the field as an argument. true
