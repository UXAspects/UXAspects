To create a toggle switch use the toggleswitch directive with an ng-model attribute to set and store the value. Use ng-disabled to control the disabled state.

Add the class toggleswitch-lg to use the large variant of the toggle switch.

The following attributes can be set to customize the behavior of the control.

Name Type Binding Description Optional
ng-model boolean variable Initially sets the value of the swtich and will be updated to contain the current value if a user toggles the switch. false
ng-disabled value variable If this value is set to true the toggle switch will be disabled. true
clickable boolean variable If set to false the switch will not be updated when clicking on it, can be used if something else is updating the state of the switch. true

Custom Styling

Above is an example of a custom style applied to the toggle switch using CSS. Using the classes described below, one may modify the appearance, size, and animations associated with the toggle switch.

The control consists of the following components that may be styled.

  • .el-toggleswitch-label is the container of both the toggle switch and the associated label.
    • .el-toggleswitch contains the toggle switch components. It will also have additional classes based on the state of the control. .checked will be present when the control is in the checked state. .disabled will be present if the ng-disabled expression evaluates to true.
      • .el-toggleswitch-bg is the visible backdrop of the toggle switch.
      • .el-toggleswitch-nub is the switch component which moves when changing state.
    • .el-toggleswitch-content contains the text label.
