Expanding textareas provide the same functionality as regular textareas however with the additional feature that they can grow in height as more content is added to them.

To create an expanding textarea use the expanding-textarea directive.

Additional attributes can be specified to configure the behavior of the textarea:

Name Type Binding Description Optional
disable-return boolean literal If true this will prevent the user from being able to take a new line by pressing the return key. true
max-visible-lines number literal Once the textarea has this many lines it will no longer grow and instead show a scrollbar. A value of 0 will grow the textarea regardless of the number of lines. true
