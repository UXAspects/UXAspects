To create a checkbox use the checkbox directive with an ng-model to set and store the value.

The following attributes can be set to customize the behavior of the control.

Initially sets the value of the checkbox and will be updated to contain the current value if a user toggles the checkbox. If ng-model is set to the indeterminate value specified using this attribute, it will neither display the checkbox as checked or unchecked, and will instead show the indeterminate variation. If set to true the checkbox will be displayed without a border and background. If set to false the checkbox will not be updated when clicking on it, can be used if something else is updating the state of the checkbox.
