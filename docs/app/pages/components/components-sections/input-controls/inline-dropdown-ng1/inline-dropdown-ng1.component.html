Inline dropdowns adjust their width according to the length of the selected item. The text preceding the dropdown can be provided using a span with the .inline-dropdown-text class.

On the div.btn-group set the dropdown attribute. The button should have the .btn, .inline-dropdown and .dropdown-toggle classes.

If you do not require the dotted line under the selected dropdown you can add the .no-line class to the button.

The dropdown can either be opened by setting the is-open attribute, or by setting the dropdown-toggle attribute on the .btn. Set on-toggle="yourFunction(open)" for yourFunction to be executed every time the dropdown changes state.
