Static tooltips can be added to elements to give the user detailed instructions on the purpose of the control when they first navigate to the page.

Add the static-tooltip directive to an element to show a static tooltip.

The following attributes can be used to configure static tooltips:

Specifies the text to be displayed within the tooltip. If specified will set the text of the dismiss all static tooltips option that will be displayed on hover. Specifies the direction the tooltip will point to the chosen element. Available options are: up, down, left and right. Specifies where the callout should be placed on the tooltip. Available options are: start, middle and end. Specifies the number of pixels to the left or right that the tooltip should be offset. Specifies the number of pixels up or down that the tooltip should be offset. If specified will set the z-index of the tooltip. If specified will allow a custom class to be set on the tooltip offering more customization. If set to true this tooltip will not be displayed initially Can be used to set a custom id on the tooltip, these must be unique for each tooltip.

The Static Tooltip Service, $staticTooltip, allows you to show and hide all tooltips or individual tooltips and check if any tooltips are visible. The full API for this service is as follows:
