The Tree View component can be created by using the tree-root tag. It allows tree content to be displayed in a familiar Explorer-like format. The component is highly extensible and supports the following features:

  • Custom Templates
  • Drag & Drop
  • Filtering
  • Lazy Loading
  • Virtual Scrolling

The data to be displayed in the Tree View should be set by using the nodes input on the component. Each node should have a name property which will define the text displayed in the node.

A node may have child nodes by adding a children property to the node object. Additional custom properties can also be added to a node object, for example an id.

Full documentation on how to implement all the features listed above, and more, can be found on the component's website.
