This example demonstrates asynchronous loading in the tree grid. The data attribute is bound to a function which returns a promise for the child items of the provided data item.

The tree-data attribute is used to examine and control the tree grid. The buttons above the grid use it to locate all “folder” type rows, and invoke the contract, expand or reload functions on them. It is also watched for changes made by the controls in the Priority column, and an array of the high priority items is maintained in the controller.
