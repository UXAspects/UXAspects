Flippable Cards are a neat and tidy way to show more information about something, without taking up screen space.

To create a Flippable Card you must use the flippable-card directive. Within this directive you will also make use of 2 other directives; card-front and card-back.

Once you have set up the directives, all you have to do is insert the content/HTML you wish to have on each side of the card. This content will be placed within the card-front and card-back tags.

There are some optional attributes you can specify on the flippable-card directive to customize it to suit your needs. These include:

Name Type Binding Description Optional
flip-style string literal Describes whether the card should flip horizontally or vertically. Defaulted to horizontal unless set to vertical. true
flip-on string literal Describes the event which will cause the card to flip. Options are hover and click, defaulted to hover.

If you specify click then you can also set a flip-on-click attribute on the element you wish to flip the card when clicked, otherwise it will flip when the card is clicked. 		true
card-width number literal Describes the width (in pixels) of the card. The default value for this is 280. true
card-height number literal Describes the height (in pixels) of the card. The default value for this is 200. true
