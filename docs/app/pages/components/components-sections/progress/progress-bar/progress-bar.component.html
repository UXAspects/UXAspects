A progress bar can be created by using the ux-progress-bar component. Custom content can be added to the bar by adding HTML inside the ux-progress-bar tag.

Additional contextual information can be provided to screen readers by adding an aria-valuetext attribute to the ux-progress-bar.

The following attributes can be used to configure the appearance and behavior of the progress bar:

The value the bar should represent. This will be compared to the max value to determine the width of the bar. The lower limit of the bar. The upper limit of the bar. The color of the track. The color of the bar.
