The as-split component can be used to create resizable panels. It should contain several as-split-area components which define the content to be displayed in each panel. The as-split component is designed to fill 100% of the height of its parent element, so it is recommended to set an explicitly defined height on the containing element.

To add accessibility features to the splitter also import the AccessibilityModule.

Full documentation on how to use the component, and more, can be found on the angular-split website.

The as-split component also has support for the following features:

  • Nested Splitters
  • Layout Switching
  • Animated Collapsible Panels
