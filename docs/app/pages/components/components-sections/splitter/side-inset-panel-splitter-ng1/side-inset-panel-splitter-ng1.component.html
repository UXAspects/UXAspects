The splitter supports a toggle button to collapse the panels in the style of the Side Inset Panel. To add the toggle button, define your splitter panels as described above but add the splitter-side attribute to the side panel, and the splitter-main to the main panel.

An on-toggle attribute can be added to the splitter to specify a function to be called when the inset panel is expanding/collapsing.

A disabled attribute can be added to the splitter to specify if the toggle button is disabled. Setting this to true will set the disabled style and make the button unclickable.
