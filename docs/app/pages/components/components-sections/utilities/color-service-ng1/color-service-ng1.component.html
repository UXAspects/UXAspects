The $colorService can be used to easily get the value for a color in the Color Palette.

This method will accept a string parameter for a color from the Color Palette and return a ThemeColor object.

The following functions can be used on the ThemeColor object:

This method will set the red value of the ThemeColor. The valid range is 0-255. This method will set the green value of the ThemeColor. The valid range is 0-255. This method will set the blue value of the ThemeColor. The valid range is 0-255. This method will set the alpha value of the ThemeColor. The valid range is 0.0-1.0. This method will return the RGB value of the ThemeColor in the following arrangement: rgb(0, 167, 162). This method will return the RGBA value of the ThemeColor in the following arrangement: rgb(0, 167, 162, 0.5). This method will return the hexadecimal value of the ThemeColor in the following arrangement: #00a7a2.
