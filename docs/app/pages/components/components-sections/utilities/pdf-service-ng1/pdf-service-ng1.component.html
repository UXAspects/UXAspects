PDF documents can be created to display table data using the $pdf service. By supplying the column definitions and a dataset a table can automatically be created and a PDF can then be generated. Currently the service is limited to creating PDFs from tables of rows and columns consisting of alphanumeric text only.

To create a document call the createTable function on the $pdf service. The createTable function can take three parameters, the first an array of column definitions (which are explained below), the second containing the data for each row and the third defining any additional configuration.

Column definitions should be an array of objects, each object can contain the following properties:

Should contain the text in the column header. Should either specify the property name in the row object containing the value or a function that will be passed the row data and should return the value to display in that column. Specifies the width of the column. You can specify sizes as a pixel value (100) or a percentage ('30%'). Additionally you can let the service determine the size using 'auto' which will take only the size required to display the data in the column, or '*' which will stretch to fill up any available space.

You can also provide several options to customize the PDF being generated. If not specified then default values will be used.

Can specify the document page size. The following sizes are available:

  • '4A0', '2A0', 'A0', 'A1', 'A2', 'A3', 'A4', 'A5', 'A6', 'A7', 'A8', 'A9', 'A10'
  • 'B0', 'B1', 'B2', 'B3', 'B4', 'B5', 'B6', 'B7', 'B8', 'B9', 'B10'
  • 'C0', 'C1', 'C2', 'C3', 'C4', 'C5', 'C6', 'C7', 'C8', 'C9', 'C10'
  • 'RA0', 'RA1', 'RA2', 'RA3', 'RA4'
  • 'SRA0', 'SRA1', 'SRA2', 'SRA3', 'SRA4'
  • 'EXECUTIVE', 'FOLIO', 'LEGAL', 'LETTER', 'TABLOID'
Can specify the orientation of the page. The possible values are portrait or landscape. Can specify the spacing on each side of the document. The array should have four values indicating the spacing (in pixels) on the left, top, right & bottom of the page. Can specify the document metadata. The available properties are:
  • title - The title of the document.
  • author - The author of the document.
  • subject - The subject of the document contents.
  • keywords - Relevant keywords for the document.
