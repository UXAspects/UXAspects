The Draggable Cards control is created using the draggable-cards directive. It allows a series of panels to be displayed in a vertical list. Panels can be selected or reordered, with the ability to add edit or remove functionality.

The draggable-cards directive should contain card elements. There are two types of card element: fixed-card and draggable-card.

A fixed-card can have a title and a subtitle only and cannot be moved. The fixed-card elements are grouped together at the top of the list.

A draggable-card can have also have a title and a subtitle. Additionally, they will have reorder controls on the left side of the card. Edit and remove buttons will also be added if functions are specified for these events. The main content of the card should be specified inside the draggable-card element.

Options

The draggable-cards directive can have the following attributes to configure it:

Can be set to 'left' or 'right'. This will change the side the callout will display on a selected card. If specified, this will show or hide the reordering controls on the card. If specified, this will show or hide the edit button on a card. Note, the edit button will not appear if an edit function has not been specified. If specified, this will show or hide the remove button on a card. Note, the remove button will not appear if a remove function has not been specified. If specified, this function will be called when a card is clicked. The function should accept one parameter which will contain the scope of the selected card. If specified, this function will be called when the remove button is clicked on a card. The function should accept one parameter which will contain the scope of the card. In this function you should remove the item from the array of draggable cards in the controller (see sample code). The remove button will not appear on any cards if this has not been set. If specified, this function will be called when the edit button is clicked on a card. The function should accept one parameter which will contain the scope of the card. The edit button will not appear on any cards if this has not been set. If specified, this function will be called when a card is moved. The function should accept one parameter which will contain an array of the card scopes in their new order.
