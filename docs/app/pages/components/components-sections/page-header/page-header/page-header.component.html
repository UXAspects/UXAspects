The ux-page-header component can be used to easily provide navigation for your application. It can display a page title, application logo, breadcrumbs, icon menus and a navigation menu with support for dropdowns and nested dropdowns. The component can be displayed in both regular and condensed forms.

The appearance and behavior of the component can be configured using the following attributes:

The path to an image to display within the top left part of the header. The list of items to display in the navigation menu. The breadcrumbs to display above the page header. The title of the current page. Defines how the navigation menu should be positioned horizontally. Determines whether or not to display the page header in the regular or condensed form. The list of icon menus to display in the top right area of the page header. Determines whether or not a back button should be visible in the page header. The event fired when the back button is clicked.

The following interfaces are used within the configuration of the page header:

The icon from the UX Aspects iconset to display in the navigation item. The text to be displayed in the navigation button. Function that will be called when the navigation button is clicked. If specified, a dropdown will be displayed containing additional items for the user to select. If set to true, the item will be disabled.
The text to be displayed in the dropdown item. Function that will be called when the dropdown item is clicked. If specified, a nested dropdown will be displayed containing additional items for the user to select (maximum of 1 nested level). If set to true, the dropdown item will be disabled.
The text to be displayed in the breadcrumb. Performs navigation to a router link if that breadcrumb is clicked. Performs navigation to a fragment if that breadcrumb is clicked. Allows you to pass query parameters to a route when a breadcrumb is clicked. This function will be called when a breadcrumb is clicked.
The name of an icon from UX Aspects Icons. Description of the purpose of the icon menu, including its badge, for assistive technologies. Content to display in a raised badge over the icon. Action to execute when the icon is clicked. Items to display as a dropdown menu.
The icon from the UX Aspects iconset to display in the dropdown item. The text to display in the dropdown item. The text to display on the right side of the dropdown item. Determines if this item should be displayed as a dropdown header. If true a divider will be placed beneath this dropdown item. This function will be called when a dropdown item is clicked.

Additionally you can add custom menu items to the top right of the page header by using the uxPageHeaderCustomMenu structural directive as content of the ux-page-header component. This is used to define templates for menu items and you can add custom click behavior.
