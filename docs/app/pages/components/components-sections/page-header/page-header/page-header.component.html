The ux-page-header component can be used to easily provide navigation for your application. It can display a page title, application logo, breadcrumbs, icon menus and a navigation menu with support for dropdowns and nested dropdowns. The component can be displayed in both regular and condensed forms.

The appearance and behavior of the component can be configured using the following attributes:

Name Type Description
@Input logo string The path to an image to display within the top left part of the header.
@Input items PageHeaderNavigationItem[] The list of items to display in the navigation menu.
@Input crumbs Breadcrumb The breadcrumbs to display above the page header.
@Input header string The title of the current page.
@Input alignment left, center, right Defines how the navigation menu should be positioned horizontally. Default center.
@Input condensed boolean Determines whether or not to display the page header in the regular or condensed form.
@Input iconMenus PageHeaderIconMenu[] The list of icon menus to display in the top right area of the page header.
@Input backVisible boolean Determines whether or not a back button should be visible in the page header.
@Output backClick EventEmitter The event fired when the back button is clicked.

The following interfaces are used within the configuration of the page header:

PageHeaderNavigationItem

Name Type Description
icon string? The icon from the UX Aspects iconset to display in the navigation item.
title string The text to be displayed in the navigation button.
select (item: PageHeaderNavigationItem) => void Function that will be called when the navigation button is clicked.
children PageHeaderNavigationDropdownItem[]? If specified, a dropdown will be displayed containing additional items for the user to select.

PageHeaderNavigationDropdownItem

Name Type Description
title string The text to be displayed in the dropdown item.
select (item: PageHeaderNavigationDropdownItem) => void Function that will be called when the dropdown item is clicked.
children PageHeaderNavigationDropdownItem[]? If specified, a nested dropdown will be displayed containing additional items for the user to select (maximum of 1 nested level).

Breadcrumb

Name Type Description
title string The text to be displayed in the breadcrumb.
routerLink string? Performs navigation to a router link if that breadcrumb is clicked.
fragment string? Performs navigation to a fragment if that breadcrumb is clicked.
queryParams string? Allows you to pass query parameters to a route when a breadcrumb is clicked.
onClick (event: MouseEvent) => void This function will be called when a breadcrumb is clicked.

PageHeaderIconMenu

Name Type Description
icon string The icon from the UX Aspects iconset to display.
badge string? | number? The text/number to be displayed in a badge beside the icon.
select (menu: PageHeaderIconMenu) => void This function will be called when the icon is clicked.
dropdown PageHeaderIconMenuDropdownItem[]? The items to display in the dropdown menu that will appear when the icon is clicked.

PageHeaderIconMenuDropdownItem

Name Type Description
icon string The icon from the UX Aspects iconset to display in the dropdown item.
title string? | number? The text to display in the dropdown item.
subtitle string? The text to display on the right side of the dropdown item.
header boolean? Determines if this item should be displayed as a dropdown header.
divider boolean? If true a divider will be placed beneath this dropdown item.
select () => void This function will be called when a dropdown item is clicked.

Additionally you can add custom menu items to the top right of the page header by using the uxPageHeaderCustomMenu structural directive as content of the ux-page-header component. This is used to define templates for menu items and you can add custom click behavior.
