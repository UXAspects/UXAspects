UI Router is a routing framework for Angular, which allows organizing parts of the user interface into a state machine.

The power of UI router lies in its ability to nest views and states.

Add ui-view directive inside the body tag.

Add a tags for navigation.  Adding ui-sref directive helps with state transitions and auto-generates the href attribute of the element itâ€™s attached to, if the corresponding state has a URL.

A navigation entry is highighted when the state is selected and is contained in the active, selected classes.

Set up the states with$stateProvider.

State parameters are discussed below:

Name Description
url Defines the path provided in ui-sref.
templateUrl Configures a state's template, e.g. physical path of the page.
abstract Indicates that a state can have child states but can't be self activated.
template Simplest way to set your template.
data Sets custom data, e.g. pageTitle.
controller Assign a controller to the state, e.g. SideNavigationCtrl as sn, pageTitle.
