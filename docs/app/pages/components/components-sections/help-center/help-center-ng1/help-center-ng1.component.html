Help Center Menu is displayed on the right side of the standard header. The Help Center menu icon drops down to a provide a general help center and page specific help.

It is contained in the help-center-menu directive. The attributes of this directive are:

Defines the Help Center URL. Defines the Help Center text for the general Help Center. Defines the icon that should be displayed to represent the help Center. Defines where to open the URL. The available options are: _self, _blank, _parent, _top, framename.

Help Center Menu consists of items that can be populated using the help-center-item attribute in each element. Any element with this attribute can define three attributes:

Defines the title of the menu item for a specific page. Defines the url to which the page navigates when this menu item is clicked. Defines a sort function (similar to sort function in JS Array.prototype) based on which the menu items are displayed.
