Tabs are definied inside a tabset element using a tab element. Use the heading attribute on the tab element or the tab-heading element to define the tab heading, along with the .tab-title class. The tabset type can be assigned tabs minimal-tab value to get the minimal tab. Set the active attribute to a property which is updated with whether the tab is active or not. You can use the disabled attribute in a similar manner. Set the select attribute to an expression that will be evaluated when the tab is selected. The align attribute can be set to text-center to align the tab heading to center. For the old style of tab strip, set the .outline-tab class on the tabset element instead of setting the tab type.
