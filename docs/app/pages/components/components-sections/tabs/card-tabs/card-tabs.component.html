Card tabs are defined inside a ux-card-tabset component using a ux-card-tab component. Inside the ux-card-tab component insert any content you want to display when the tab is selected.

Also inside the ux-card-tab component an element with a uxCardTabContent directive should be defined which will contain the content that will be shown in the card.

If the ux-card-tabset has more cards than can be displayed in the space available arrow buttons will appear to allow the user to scroll through all the cards.

Defines whether the tab content should appear above or below the cards. The possible values are top and bottom.
