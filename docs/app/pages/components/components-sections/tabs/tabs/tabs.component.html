The ux-tabset component can be used to split up content into different tabs. It should contain one or more ux-tab components as children.

Each ux-tab should provide a tab heading. This can either be a string specified using the header attribute, if you want further customization you can add an ng-template as a child of the ux-tab with a uxTabHeading attribute. The content of this template will be placed in the tab heading.
