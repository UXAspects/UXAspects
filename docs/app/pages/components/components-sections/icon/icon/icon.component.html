The ux-icon component can be used to dynamically display icons and allows easy control of their size and orientation. Icons can be selected from the UX Aspects ux-icons set. The ux-icon component also allows alternative icons to be loaded from a custom icon set to override default icons. For example, icons for a specific project could be implemented via a custom set, or an icon set created for a particular size could be used

The following properties can be used to configure the component:

The name of the icon to be displayed. By default this will be one of the UX Aspects icons, unless a custom icon with this name has been defined. The size of the icon to be displayed. This value is in CSS units for example 24px or 2rem. Rotation of the icon in degrees. The icon can be horizontally flipped if set to true. The default is false.” The icon can be vertically flipped if set to true. The default is false.”
