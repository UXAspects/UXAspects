The timepicker can be created using the timepicker element. The ng-model should be set to the Date object that provides the time state. The hour-step and minute-step attributes indicate the number of hours or minutes to increase or decrease when using a button respectively. The change-total-time attribute can be provided a boolean value which decides whether or not to update the entire clock when an individual parameter is updated. A callback function can be provided to the ng-change attribute which will be called when the model changes. ng-disabled can be used to control the disabled state of the time picker.
