The ux-time-picker component can be used to select the time component of a Date object.

The following options can be used to configure the appearance and behavior of the component.

The value to display. Whether to show the hour selector. Whether to show the minute selector. Whether to show the second selector. Whether to show the meridian (AM/PM) selector. If this is false, the 24-hour clock will be used. Whether to show increment and decrement buttons in the time picker. The number of hours to increment or decrement by when using the spinner buttons, arrow keys, or mouse scroll wheel. The number of minutes to increment or decrement by when using the spinner buttons, arrow keys, or mouse scroll wheel. The number of seconds to increment or decrement by when using the spinner buttons, arrow keys, or mouse scroll wheel. The minimum value that the component will allow. The maximum value that the component will allow. An array containing the labels to show in the meridian selector. Whether the arrow keys can be used to increment or decrement the selected time component. Whether the mouse scroll wheel can be used to increment or decrement the selected time component. Whether the control is disabled. Emitted when the value changes. Emitted when the validity of the control changes.
