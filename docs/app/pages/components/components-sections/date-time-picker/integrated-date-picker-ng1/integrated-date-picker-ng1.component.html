The integrated date picker can be used to pick date, time and timezone within a single component. This is defined as an input element having the date-time-picker-popup attribute.

The ng-model attribute should contain an object with date and time values. The timezones are passed as an array to the timezones attribute, with the default index given to the timezone-default-index attribute. The resulting date, time and timezone will be set to the object passed to the ng-model.

The component can be configured with the following options provided as attributes:

Name Type Binding Description Optional
ng-model object variable Should contain a json object with date, time and timezone values. false
is-meridian boolean variable Indicating whether the time picker should work in 12 hour mode or 24 hour mode. The 12 hour mode is selected when is-meridian is set to true and the 24 hour mode is selected when is-meridian is set to false. true
timezones array variable An array of values containing the timezones. true
timezone-default-index number variable The array index of the default timezone to be selected. true
choose-time boolean variable Indicates whether time picker should be displayed. true
choose-date boolean variable Indicates whether date picker should be displayed. true
choose-seconds boolean variable Indicates whether the seconds field in the time picker should be displayed. true
choose-timezone boolean variable Indicates whether timezone picker should be displayed. true
default-day-time string literal Choose whether the default time should be the start or end of the day. Valid values are start (default) and end. true
validator function variable Set a funtion here to override the default validation checks with your own custom checks. This should return true if the validation passes; otherwise return false. true
placement string variable Defines whether the popover should appear to the bottom or top of the input. The valid options are top and bottom. The default value is bottom. true
display-format-12hr string variable Sets the format for how the date will appear in the input box when in 12hour mode. The default format is ddd MMM DD, YYYY h:mm:ss A. true
display-format-24hr string variable Sets the format for how the date will appear in the input box when in 24 hour mode. The default format is ddd MMM DD, YYYY HH:mm:ss. true

Note: Refer to the moment.js documentation for more information about format strings.

The time picker only mode is created by setting the choose-time attribute to true and the choose-date and choose-timezone attributes to false in the integrated date picker. The hpe-alarm class can be used to set the clock icon.
