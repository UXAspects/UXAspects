Create a div.input-group with the span.input-group-addon containing the .hpe-icon.hpe-calendar icon. On the span.input-group-addon you must set ng-click="vm.open($event)".

To add keyboard functionality you must set ng-keypress="vm.openKey($event)" on the span.input-group-addon. The open and openKey functions are set on the controller and are responsible for opening the datepicker popup.

On the input you must specify ng-model with the date value, datepicker-popup which can have the format string for the input specified or left blank. Like the span.input-group-addon the input requires ng-click="vm.open($event)" as well as is-open="vm.opened".

The datepicker requires an explicit date validator and is defined as ng-blur="vm.change()". The “change” method is defined in the controller and is responsible for validating the date.

When there is no label for the control, include an aria-label on the input element to describe the date for assistive technologies.

Example code for a “change” method has been provided to show how to format this function.

Here are some more attribute options you can specify for the datepicker:

The minimum date that can be chosen. By default there is no limitations. The maximum date that can be chosen. By default there is no limitations. The current mode of the date picker (day|month|year). A function thats returns a boolean as to whether a date is disabled or not. Function is passed in the date then the mode e.g. date-disabled="isDateDisabled(date,mode)".
