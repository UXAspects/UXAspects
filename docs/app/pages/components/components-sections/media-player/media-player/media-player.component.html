The media player component can be used to view video and audio files. It supports a range of the most popular video and audio file formats. A comprehensive list of supported formats in each browser can be found here.

To add a media player to your page use the ux-media-player selector. The following attributes can be used to configure the appearance and behavior of the media player:

Name Type Description
@Input type string Defines the appearance of the media player. The two possible values are video and audio. The media player will adapt it's appearance to best suit the type specified.
@Input source string The url to the media file to be loaded by the media player.
@Input quietMode boolean If enabled, the controls in the media player will be hidden unless the mouse is over the player and will appear in a darker style. Dark mode is automatically enabled in full screen mode. Quiet mode is only available for videos.
