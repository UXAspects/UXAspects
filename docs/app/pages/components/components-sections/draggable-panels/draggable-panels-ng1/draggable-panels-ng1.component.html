Draggable Panel Container

Draggable functionality can be implemented by adding a draggable-panel-container directive to the page. This will act as a container for any draggable-panel directives.

To specify the size of the draggable-panel-container you can apply a class that will set the width and height, such as .col-xs-6 as in the sample code above.

It is possible to have multiple draggable-panel-container elements on one page. By default any draggable-panel can be moved to any draggable-panel-container, however this can be restricted.

If a draggable-group attribute has been added this will limit which draggable-panel elements can be moved to which draggable-panel-container. Any draggable-panel-container elements that share the same draggable-group can move draggable-panel elements between each other.

Events

You can also add the following attributes to a draggable-panel-container directive to specify which function to be called when certain events occur.

Name Type Binding Description Optional
on-drag-start function variable Will be called when the user first begins to move a panel. true
on-drag-over function variable Will be called when the user moves a panel over a draggable-panel-container. true
on-drag-end function variable Will be called when the user finishes dragging a panel. true

Each function should expect two arguments:

  • event - event, contains the mouse event that was triggered.
  • ui - object, contains information about the panel being dragged.
    • item - object, represents the current dragged element.
    • offset - object, absolute position represented as {{'{'}} top, left }.
    • position - object, current position represented as {{'{'}} top, left }.
    • originalPosition - object, original position represented as {{'{'}} top, left }.
    • sender - object, the source draggable-panel-container if moving a draggable-panel from one draggable-panel-container to another.
    • placeholder - object, the element being used as a placeholder.

Draggable Panels

To identify which elements should be draggable use the draggable-panel directive. The draggable-panel directives should be placed within the draggable-card-container they will begin in.

The draggable-panel directive should contain a single div element which will become the draggable panel. This div can be styled in any way and have child elements.

Drag Handle

An element within the draggable-panel directive should be given the drag-handle attribute. This will identify the element that the user must mouse down on to drag the panel. If the user should be allowed to drag anywhere within the panel simply apply the attribute to the div immediately inside the draggable-panel.
