Hierarchy Bar defines a hierarchy of nodes in any component which are contained inside a hierarchy-bar element. The popover in the hierarchy bar lists the child nodes and when clicked will highlight that node.

A hierarchy-bar element has the following attributes:

Property Type Binding Description Optional
data array of objects variable The data the hierarchy bar should display. This array of objects contains a node and its parent and can contain any properties, however all possible child items should be listed in an array under a children property. The first item in the array should be the root node with the selected child node as the next item in the array. false
options object variable Allows you to configure the appearance and behavior of the hierarchy bar. All available options can be found in the table below. false
select-node function variable A function expression with parameters (function signature is sufficient) which is called when a hierarchy item (node) in the hierarchy bar is clicked. This could be used to update the hierarchy bar or perform actions on the element. false
container-class string literal Provides the container class name. The hierarchy bar will clip/overflow based on the container dimensions (re-size). true

HierarchyBar Options

Property Type Description
enabled boolean Specifies whether or not the hierarchy bar will be shown (default: true).
overview function If specified, a button on the left of the hierarchy bar will be shown. When clicked this function will be called. The purpose of this would be to allow you to present more information regarding the tree or a list of all items in the chart.
image function If specified, this function will be called and passed a node. It should return the url to an image if one should be shown beside the hierarchy bar.
valueFormatter function If specified, this function will be called and passed a node. It should return the text to display in the hierarchy bar. By default it looks for a name property.

Action

Property Type Description
title string Text to display along side the activity indicator displayed when data is loading.
event function A function that is called when an arrow is clicked. It will be passed the associated breadcrumb as a parameter. To display a loading indicator in the dropdown have the function return a $q.defer().promise. The loading indicator will be displayed until the promise is resolved.

Note: Refer to the Organization Chart page for an example of the hierarchy bar with popovers on top of the chart.
