The select-table component can be used for single select in a table format. The component can be defined using a select-table element. The options for the select component should be passed as attributes as explained below:

Name Type Binding Description Optional
values array variable Array of strings or Array of objects, this attribute should be passed with the values to be shown in the select table. false
select-key string variable When the options are passed as an array of objects, this attribute should be set to the name of the key, whose value from the object will be displayed. Note that this should only be set when the options are passed as an array of objects, and not required when the options are passed as an array of strings. true
selected string variable The selected option will be set to the variable passed to the selected attribute. false
search-text string variable The component can be associated with a search text which can filter the options. The search-text attribute can be set to the text with which options will be filtered. false
table-height string variable The height of the table can be changed by providing a string to the table-height attribute (e.g. "50%"). This can be set to px, % etc. false
select-hidden-items string variable Controls the behaviour when selected items are filtered from view. By default filtered items remain selected. Set to the string 'clear' to clear the selected item when the search-text filter has removed it from the list of visible items. Alternatively, set to 'reselect' to both clear the selected item, and reselect it when it becomes visible again. true
