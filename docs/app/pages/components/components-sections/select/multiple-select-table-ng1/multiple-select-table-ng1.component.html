The Multiple Select Table can be used for selecting multiple items in a table format. As with the Single Select Table described above, it also makes use of the select-table component. The only additional feature is the multiple-select attribute which is used solely for selecting multiple items. All other attributes for the select-table maintain the same functionality with the exception of the following:

Allows selection of multiple items. If set to true the user will be able to select and deselect multiple items. This should be set as an array for the Multiple Select Table as there can be multiple selected values, whereas the Single Select Table only takes one selected value. It can be an array of strings or an array of objects. If it is an array of objects, the select-key attribute (as described in the Single Select Table) should be set to the name of the key, whose value from the object will be displayed. For example, if you have an object like: {{codeSnippet}} and you want the name to be displayed, you would set select-key="name". This attribute is not used in the Multiple Select Table.
