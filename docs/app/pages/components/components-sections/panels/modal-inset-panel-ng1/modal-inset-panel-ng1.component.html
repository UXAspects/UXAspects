The modal inset panel provides the functionality to have additional information appear in an animated slide out panel within a modal. This can be used to display more detailed information or to contain controls for obtaining information.

The modal-inset-panel directive should be placed at the top of the appropriate modal body element. In the example below it is placed in the div.modal-body however this may differ slightly depending on the type of modal being used.

The following attributes can be specified to configure the behavior of the modal-inset-panel.

This value should be changed to show or hide the panel. Sets which side of the modal the panel should appear on. The available options are right or left. Sets the desired width of the panel in pixels. If specified, this will be called when the panel has been dismissed.

Modal Inset Panel Contents

To specify the contents of the inset panel use the modal-inset-header and modal-inset-content directives. These should be added inside the modal-inset-panel directive.

The modal-inset-header directive will add a header to the panel that contains a close button to dismiss the modal inset panel. If set to false the close button will be hidden and the visibility of the panel must be handled manually.

The modal-inset-content directive should act as a container for the main content of the panel. The height of the panel will always fill the modal body, however if the content may be larger than the available space consider adding a scrollbar.
