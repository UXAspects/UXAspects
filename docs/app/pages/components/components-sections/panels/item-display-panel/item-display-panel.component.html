To add the item display panel component to your application you need to import the ItemDisplayPanelModule which allows you to use the ux-item-display-panel component.

The following attributes can be used to customise the behaviour of the component.

Defines whether the item display panel is visible. The header displayed at the top of the item display panel. The number of pixels from the top of the screen, this can be used to keep the item display panel below the page header. Defines whether the item display panel will animate in and out from the side of the screen. Defines whether the item display panel will have a rounded shadow visible under the header. Defines whether the item display panel will have a box shadow. Defines whether the item display panel will be inline with the page content as opposed to coming in from the side of the screen . Defines whether the item display panel should prevent closing when clicking outside the panel. Defines whether the close button should be visible on the item display panel. If using two-way binding this will update when the item display panel is shown and hidden.

To add the main body of content to the item display panel the uxItemDisplayPanelContent directive can be added as an attribute allowing you to add any required content below the header.

To add a footer to the item display panel the uxItemDisplayPanelFooter directive can be added as an attribute allowing you to add any required content to the bottom of the item display panel
