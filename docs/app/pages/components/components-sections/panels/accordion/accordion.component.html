To create collapsible panels use the accordion tag. This contains the closeOthers property, which if set to true will close the other panels, and if set to false will close only the individual panel.

Add the required number of panels underneath by adding accordion-group elements. These have a heading property to provide the heading for that panel. An accordion-group element can also have an isOpen property, which if set to true will show open the panel initially. The panel content should be placed within the accordion-group element.
