The ux-accordion component can be used to create collapsible panels. To display chevrons in the accordion header add the accordion-chevron class to the ux-accordion-panel element.

If set to true, only one panel can be expanded at any given time. When a panel is expanded all other will collapse. Sets the text to display in the panel heading. If set to true the panel will display in the expanded state. If set to true the expand/collapse functionality be be disabled. This event will emit when the expanded state changes.

To further customize the header content an ux-accordion-panel-header element may be inserted as a child of a ux-accordion-panel. Inside this element you can place the HTML to display in the panel header.
