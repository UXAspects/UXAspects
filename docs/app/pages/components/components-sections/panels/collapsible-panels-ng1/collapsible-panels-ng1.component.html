To get collapsible panels create an accordion tag. This contains the attribute close-others, which if set to true will close other panels, and if set to false will close only individual panels.

Add the required number of panels underneath by adding accordion-group elements. These have attribute heading to provide the heading for that panel. An accordion-group element can also have an is-open attribute, which if set to true will keep the panel open initially. The accordion-group can have the panel-body content within the tags.
