Floating Action Button can be opened in any of the 4 directions (top, bottom, left or right).

Floating Action Button - Horizontal (Right)

Floating Action Button - Vertical (Up)

Floating Action Button - Horizontal (Left)

Floating Action Button is contained in floating-action-button directive.

The attributes defining this directive are:

An array of objects with the following properties:
  • icon - The icon to display in the button. This should be an icon from the icon set.
  • event - A function that will be called when the button is clicked.
  • tooltip - If defined, a tooltip will display when the button is hovered.
  • tooltipPlacement - If defined, the tooltip will be displayed in the specified position. Possible values are top, right, bottom or left.
Defines the direction in which the child button should expand (top, bottom, left, right). If defined, a tooltip will be displayed when the user hovers over the floating action button.
