Floating Action Button can be opened in any of the 4 directions (top, bottom, left or right).

Floating Action Button is contained in floating-action-button directive.

The attributes defining this directive are:

Array of icons and events, that defines the child buttons shown when the primary button is expanded. Each element of the array should contain an icon property which should be the name of an icon from the icon set. There should also be an event property which should reference a function that will be called when a button is clicked.
Additionally, each item can have a tooltip property which, if defined, will display a tooltip when the button is hovered. The tooltip position can also be specified by adding a tooltipPlacement property with a value of top, right, bottom or left. Defines the direction in which the child button should expand (top, bottom, left, right). If defined, a tooltip will be displayed when the user hovers over the floating action button. If defined, the tooltip will be displayed in the specified position. Valid options are top, right, bottom or left.
