Floating Action Button can be opened in any of the 4 directions (top, bottom, left or right).


Floating Action Button is contained in floating-action-button directive.

The attributes defining this directive are:

Name Type Binding Description Optional
items array variable Array of icons and events, that defines the child buttons shown when the primary button is expanded.

Each element of the array should contain an icon property which should be the name of an icon from the icon set.

There should also be an event property which should reference a function that will be called when a button is clicked. 		false
direction string variable Defines the direction in which the child button should expand (top, bottom, left, right). false
