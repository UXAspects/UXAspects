The Bootstrap pagination component can be used to create a paging control. Specify the paging parameters using the totalItems and itemsPerPage properties. The active page number is bound to ngModel. The numPages output property receives the total number of pages.

To use the UX Aspects icons in the previous and next buttons, HTML markup can be assigned to the previousText and nextText properties. See the TypeScript example code below for recommended icon bindings.
