The Bootstrap btnCheckbox directive can be used to create a button which can toggle between two states. The btnCheckboxTrue and btnCheckboxFalse properties can be used to bind custom values for the states.

The classes button-toggle-primary and button-toggle-accent provide the themed styles seen above.
