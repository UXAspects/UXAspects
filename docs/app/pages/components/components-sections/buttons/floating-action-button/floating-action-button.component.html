Floating Action Buttons are contained in a ux-floating-action-buttons component. Each action button should be added as child using the ux-floating-action-button component.

The fab-primary attribute should be added to the floating action button that will permanently be visible and will be used to display the additional buttons when clicked.

The following input can be used to customize the behavior of the ux-floating-action-buttons component:

Defines direction items should be displayed. Possible values are top, right, bottom or left.

The following input can be used to customize the appearance of the ux-floating-action-button component:

If specified, defines which icon from the icon set to display in the button. If you wish to display custom content you can simply add children to the component and they will be displayed within the button.
