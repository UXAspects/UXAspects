Split buttons are rendered as two separate buttons, one with the button text and the other drops down to open the menu.

Split button dropdowns use the .btn-group class to group the button and the dropdown icon. The menu is contained in the .dropdown-menu class.

The ngx-bootstrap dropdown functionality can be used to create the dropdown. See Dropdowns for more information.
