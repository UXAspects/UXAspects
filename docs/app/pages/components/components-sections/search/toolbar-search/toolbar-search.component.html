The ux-toolbar-search component can be used to toggle the visibility of a search input box in the context of a toolbar. Within this component, create an input field with the attribute uxToolbarSearchField, which will act as the search input box, and a button with the attribute uxToolbarSearchToggle, which will toggle the visibility of the input field. A typeahead dropdown can be added to the input field if required; alternatively, a search results page could be loaded by hooking into the (keyup.enter) event.

For use in the masthead or against any kind of dark background, apply the toolbar-search-inverse class for an inverted color scheme.

The following properties are available to configure ux-toolbar-search:

The direction in which the search box will expand.
