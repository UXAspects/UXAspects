The Search Toolbar is contained in the search-toolbar directive and the .expand-input class.

Search Toolbar takes the following attributes:

Name Type Binding Description Optional
search-typeahead array variable Name of the list where the typeahead data is stored. false
place-holder string literal Placeholder text for an input. false
close-search string literal Should be set to a value which when clicked closes the input text. false
on-search function variable Defines the function to be called when the enter key is pressed. The function will be called with the current text entered in the field as an argument. false
on-focus function variable Defines the function to be called when the search bar is focused. true
