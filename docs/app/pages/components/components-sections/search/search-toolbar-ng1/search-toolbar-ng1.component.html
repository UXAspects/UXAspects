The Search Toolbar is contained in the search-toolbar directive and the .expand-input class.

Search Toolbar takes the following attributes:

Name of the list where the typeahead data is stored. Placeholder text for an input. Should be set to a value which when clicked closes the input text. Defines the function to be called when the enter key is pressed. The function will be called with the current text entered in the field as an argument. Defines the function to be called when the search bar is focused.
