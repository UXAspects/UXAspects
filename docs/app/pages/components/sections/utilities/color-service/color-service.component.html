The ColorService can be used to get the Hex, RGB or RGBA value for a color in the Color Palette.

Name Args Returns Description
getColor string Object This method will accept a string parameter for a color from the Color Palette and return a ThemeColor object.

ThemeColor Object

The ThemeColor object is used to provide the selected color from the Color Palette

The following functions can be used on the ThemeColor object:

Name Args Returns Description
setRed number This method will set the red value of the ThemeColor. It will take a number as a parameter.
setGreen number This method will set the green value of the ThemeColor. It will take a number as a parameter.
setBlue number This method will set the blue value of the ThemeColor. It will take a number as a parameter.
setAlpha number This method will set the alpha value of the ThemeColor. It will take a number as a parameter.
toRgb number This method will return the RGB value of the ThemeColor.
toRgba number This method will return the RGBA value of the ThemeColor.
toHex number This method will return the hexadecimal value of the ThemeColor.
