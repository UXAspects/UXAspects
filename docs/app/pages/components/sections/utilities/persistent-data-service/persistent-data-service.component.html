The PersistentDataService can be used to set and get data in localStorage, Cookies or sessionStorage. The service can be used to choose which to set or will attempt to use localStorage by default, falling back to Cookies if this fails.

The PersistentDataStorageType enum is used when setting the storage type

Name Args Returns Description
setData string, string, PersistentDataStorageType? This method expects a key and value parameter as strings and can also accept a PersistentDataStorageType if you wish to set the storage type, otherwise it will follow defauly behaviour.
getData string, PersistentDataStorageType? string This method will accept a string parameter for the key and will return a string containing the value associated with that key. It will aslo accept a PersistentDataStorageType parameter which is required if this was used when setting the data.
