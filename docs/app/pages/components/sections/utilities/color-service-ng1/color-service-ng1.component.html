The $colorService can be used to easily get the value for a color in the Color Palette.

Name Description
getColor This method will accept a string parameter for a color from the Color Palette and return a ThemeColor object.

The following functions can be used on the ThemeColor object:

Name Description
setRed This method will set the red value of the ThemeColor. It will take a number as a parameter.
setGreen This method will set the green value of the ThemeColor. It will take a number as a parameter.
setBlue This method will set the blue value of the ThemeColor. It will take a number as a parameter.
setAlpha This method will set the alpha value of the ThemeColor. It will take a number as a parameter.
toRgb This method will return the rgb value of the ThemeColor.
toRgba This method will return the rgba value of the ThemeColor.
toHex This method will return the hexadecimal value of the ThemeColor.
