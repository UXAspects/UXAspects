Static tooltips can be added to elements to give the user detailed instructions on the purpose of the control when they first navigate to the page.

Add the static-tooltip directive to an element to show a static tooltip.

The following attributes can be used to configure static tooltips:

Name Type Binding Description Optional
content string variable Specifies the text to be displayed within the tooltip. false
dismiss-text string variable If specified will set the text of the dismiss all static tooltips option that will be displayed on hover. The default is 'HIDE TIPS'. true
direction string variable Specifies the direction the tooltip will point to the chosen element. Available options are: up, down, left and right. false
position string variable Specifies where the callout should be placed on the tooltip. Available options are: start, middle and end. false
offset-x number variable Specifies the number of pixels to the left or right that the tooltip should be offset. true
offset-y number variable Specifies the number of pixels up or down that the tooltip should be offset. true
z-index number variable If specified will set the z-index of the tooltip. Default value is 10098. true
tooltip-class string variable If specified will allow a custom class to be set on the tooltip offering more customization. true
tooltip-hidden boolean variable If set to true this tooltip will not be displayed initially true
tooltip-id string variable Can be used to set a custom id on the tooltip, these must be unique for each tooltip. true

The Static Tooltip Service, $staticTooltip, allows you to show and hide all tooltips or individual tooltips and check if any tooltips are visible. The full API for this service is as follows:

Name Description
showAllTooltips This function will hide all the tooltips.
hideAllTooltips This function will hide all the tooltips/
showTooltip Arguments:
  • id - string

This function will show the tooltip with the matching id.
hideTooltip Arguments:
  • id - string

This function will hide the tooltip with the matching id.
tooltipsVisible This function returns true if any tooltips are visible, otherwise returns false.
