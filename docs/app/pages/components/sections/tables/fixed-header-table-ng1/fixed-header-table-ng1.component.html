A fixed header table is provided by the scrollable-table directive. The table header stays in place while the table content is scrolled independently.

The attributes of this directive are:

Name Type Binding Description Optional
watch object variable Defines the values of the rows. false
container-height number literal Defines the height of the scrollable container. false
header-height number literal Defines the height of the fixed header. false
watchHeaders object variable Defines the values of the headers, required if using dynamically changing headers. true

Refer to the Sorting page for information about sorting the table headers.
