As part of UX Aspects we proved two common prebuilt Filter components, however you may require different filter components specifically for your application. We provide some useful classes for you to use to make the process of creating your own filter component simple while ensuring that it will work well with Filter Containers automatically.

Creating a Custom Filter Component

Your custom filter component should extend the FilterBaseComponent class. This base class will provide you with the following functions:

  • addFilter(filter: Filter): void – Will add a filter to the list of active filters.
  • removeFilter(filter: Filter):void – Will remove a filter from the list of active filters.
