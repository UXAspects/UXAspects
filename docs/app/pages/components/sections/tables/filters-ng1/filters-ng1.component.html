Filters are contained in the filter-container directive.

Each directive has an internally defined Clear option which would appear when a filter is set. The clear-tooltip attribute can be set to a string which will be displayed as tooltip for the clear button.

Filter Containers contain filters defined by the filter directive. Each filter should have a name.

Each filter has a set of options to filter data. Filter options are contained in the filter-option directive.

Filter Options should have the following attributes:

An object containing the filter option name to be displayed. A function expression called when the option is selected. This method performs filtering, pagination and updates the data count. A boolean object indicating if this is the default filter (e.g., AUTHOR) applied when Clear All button in the filter container is clicked.
