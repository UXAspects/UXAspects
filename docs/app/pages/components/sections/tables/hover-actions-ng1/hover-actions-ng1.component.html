Hover actions are very similar to list hover actions, however they are not limited to use within a table. Hover actions can be used inside any element, and once that element is hovered over the hover actions will appear and can be clicked or navigated to using the arrow key and also allow selection using the return key.

To add hover actions to an element add the hover-actions directive. This can have the following attribute:

Name Type Binding Description Optional
hover-element string literal This should specify a CSS selector that defines which parent element should be hovered to show the hover actions. If this is not specified, the immediate parent will be set as the hover element. true

To add hover actions add hover-action directives to the hover-actions directive. A hover-action can use the following attributes to configure itâ€™s appearance:

Name Type Binding Description Optional
name string literal The name of the action, placed in a tooltip that is shown when the action is hovered. false
icon string literal The icon class from the icon set. false
click function variable This expression will be evaluated when the action is clicked or enter is pressed when it has focus. false
