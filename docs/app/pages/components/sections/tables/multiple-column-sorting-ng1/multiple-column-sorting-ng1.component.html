The different options to sort are contained in the multi-sortable-header directive. The multi-sortable-header will require one of the following classes to be added in order to set the text size for the headers.

  • sorter-header-sm - sets the font size to 14px.
  • sorter-header-md - sets the font size to 16px.
  • sorter-header-lg - sets the font size to 18px.

If using this in conjunction with the scrollable table one of the following classes is required.

  • scrollable-header-sm - sets the font size to 14px.
  • scrollable-header-md - sets the font size to 16px.
  • scrollable-header-lg - sets the font size to 18px.

The following attributes will need to be supplied to the sortable header:

Name Type Binding Description Optional
headers array variable Array of objects (defined below) used to store all the required header information. false
default-sorter array variable Array of strings indicating which columns will be sorted by default when the page loads, setting this to a variable will allow the sort to be remembered between visits. false
default-order array variable Array of boolean options indicating the sort order of the default columns when the page loads, setting this to a variable will allow the sort to be remembered between visits. false

The controller implements the select function and will accept the following attributes within the headers object:

Name Type Description Optional
sorterHeader string The column header. false
sortable boolean True if the column can be used as a sorter. false
sort string Sort expression for the column. false
center boolean True if the column header should be centered. false
fixedClass string A class for the header in order to set the column to a fixed width. This results in columns that do not change size when a sort has been applied or according to data in the cells. The following classes have been created to allow for fixed column widths:
  • table-col-icon - width of 65px. Optimum width for a column containing icons.
  • table-col-xs - width of 5%.
  • table-col-sm - width of 10%.
  • table-col-md - width of 20%.
  • table-col-lg - width of 30%.
  • table-col-xl - width of 40%.
  • fixedLayout - sets the table to a fixed layout. To be used in conjunction with the classes above and should be placed within the table tag.
Note: when applying a fixedClass to a column header you must also apply that same class to the td (data row) for that column. You should also specify the class in the table tag to be fixedLayout. 		true
