Facets are contained inside a facet-container element. A facet-container element can have many facet elements underneath it. The facet-container must have the select-title and clear-text attributes set. select-title should be set to the value which appears above the selected facets. clear-text should be set to the value which appears in the clear all link. In addition to these options the no-items-text attribute can be set which will display the specified text when no facet options have been selected. no-items-text is an optional attribute.

The clear-all-fn is an optional attribute in this directive that defines a function expression called when the clearAll is clicked. This option will override the default function expression provided in UX Aspects and can be used to define a user method.

A facet element may have the following attributes:

Name Type Binding Description Optional
name string variable The facet title. false
facet-scroll string literal Set to "on" to enable a scrollbar for the facet options, or "off" to disable it. Default: on. true

A facet element can contain many facet-option elements. facet-option elements can have the following attributes:

Name Type Binding Description Optional
name string variable A bindable property which will be displayed as the option. false
count number variable An optional bindable property which will appear alongside the name in braces. Note that the count should update as facets are applied. true
select function variable A function expression which will be called when the option is selected. This function should perform any filtering that is required and update counts as necessary. The select count should update as facets are applied, based on the number of items selected which will be displayed on the toolbar in selection mode. false
deselect function variable a function expression which will be called when the option is deselected. This function should perform any filtering that is required and update counts as necessary. The select count should update as facets are applied, based on the number of items selected which will be displayed on the toolbar in selection mode. false
load-delay number variable An optional property to specify a delay in milliseconds to load the facets. true
disabled boolean variable If true the facet-option will appear disabled and clicking on it will have no effect (default: false). true
show-zero boolean variable If true the facet-option will be visible even if the count is 0 (default: false). true
selected-aria-label boolean variable A property which accepts a string used in assistive technologies to indicate whether the facet option is selected. true
simplified boolean variable By default the style of the checkboxes will be simplified, setting this to false will use the standard checkbox style. true
Property Description
updateSelectCounts() A function expression which will be called when facet option is selected (select) or deselected ( deselect). This will update the count based on the number of items that are selected in selection mode when the filtering is done on facets.
