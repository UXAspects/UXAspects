The Facet Check List component can be used to display a list of Facets that can be selected or deselected to provide the user with filtering capabilities. This component is designed to be placed within the Facet Container component.

To add a Facet Check List component to your application you need to import the FacetsModule which allows the use of the ux-facet-check-list component.

The following attributes can be used to customise the behavior of the component:

Name Type Description
@Input facets Facet[] Defines the complete list of facets that can be selected.
@Input @Output selected Facet[] This will allow you to define an initial set of selected facets. If two-way binding is used this array will get updated any time the selected facets change.
@Output events FacetEvent This will be triggered when a facet is selected, deselected or all facets are deselected. The event will be an instance of either FacetSelect, FacetDeselect or FacetDeselectAll and will contain the facet being selected or deselected in a facet property (deselect all will not contain affected facets).
@Input header string Defines the text displayed at the top of the Facet Check List.
@Input scrollbar boolean If false the list will grow to display all possible facets. If true a scrollbar will appear to prevent the list from growing too large. Default: true.
@Input expanded boolean Defines whether or not the Facet Check List should be initially expanded or not. Default: true.
