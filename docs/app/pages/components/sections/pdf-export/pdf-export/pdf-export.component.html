The PdfExportService can be used to export sections of a webpage to be saved as PDF or printed.

Name Type Description
getDocument() HTMLElement This will return the HTMLElement containing the items that will be exported only.

Items can be marked for exporting using the pdfExportItem directive. Items can be set to only be visible in the pdf view and not on the webpage by setting the directive on an ng-template.

Name Type Description
@Input row number? Defines which row the exported item should be displayed on.
@Input columns number? Defines how many columns the exported item should fill.
@Input data any? Sets the data required for items inside the ng-template.
