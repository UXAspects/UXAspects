A radio button can be created by using the ux-radiobutton component. The following attributes can be used to configure the appearance and behavior of the component:

Name Type Description
@Input @Output value any This should be a two way binding and will store the currently selected option. Each radiobutton in the same group should have the same value variable.
@Input option any This should contain the value that this radiobutton represents. This will be stored in the value variable when the radiobutton is selected. No two radio buttons should have the same option value within the same group of radio buttons.
@Input simplified boolean If set to true the checkbox will be displayed without a border and background. The default value is false.
@Input clickable boolean If set to true the checkbox will not toggle state when clicked. The default value is false.

In addition to the options listed above, a radiobutton can be disabled using the disabled attribute. You can also provide the radiobutton with a name or id if required.
