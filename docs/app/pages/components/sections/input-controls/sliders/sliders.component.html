Options

A wide range of options can used to customize the appearance and behavior of the component. These options should be stored in an object and passed to the component by adding an options attribute to the directive.

The default options for each chart are as follows:

Full details on each property and the possible values for each can be found below. All properties are optional, if they are not specified the default value will be used.

SliderOptions

Property Type Description
type SliderType Defines whether the slider should allow selection of a single value or a range of values. The two valid values are SliderType.Value and SliderType.Range. The default value for this property is SliderType.Value.
handles SliderHandleOptions This should be an object that contains properties configuring the appearance and behavior of the draggable handles on the slider. If not specified the default options will be used.
track SliderTrackOptions This should be an object that contains properties configuring the appearance and behavior of the track on the slider. If not specified the default options will be used.

SliderHandleOptions

Property Type Description
style SliderStyle This allows you to choose the appearance of the drag handles. The two available options are SliderStyle.Button and SliderStyle.Line. The default option is SliderStyle.Button.
callout SliderCallout This should be an object that configures the appearance and behavior of the drag handle callout. If not specified no callout will be shown.

SliderCallout

Property Type Description
trigger SliderCalloutTrigger This option will specify when the callout should appear. The available options are:
  • SliderCalloutTrigger.None - The callout will never appear. This is the default option.
  • SliderCalloutTrigger.Hover - The callout will appear when the mouse is over the handle and will remain visible when dragging the handle.
  • SliderCalloutTrigger.Drag - The callout will appear only when the handle is being dragged.
  • SliderCalloutTrigger.Persistent - The callout will always be visible.
background string Specified the background color of the callout. The default value is grey2.
color string Specified the text color of the callout. The default value is white.
formatter (value: number) => string | number This allows you to specify a function that will format the value in the callout. The function should accept the value as a parameter and return the modified value. By default the value will not be modified.

SliderTrackOptions

Property Type Description
height SliderSize This specifies the height of the track. There are two available values, SliderSize.Wide and SliderSize.Narrow. The default value is SliderSize.Wide.
min number Specifies the minimum value shown on the slider. The default is 0.
max number Specifies the maximum value shown on the slider. The default is 100.
ticks SliderTicksOptions Should be an object containing properties to customize the ticks shown beneath the chart. By default major ticks are shown on every 10th value with labels, minor ticks are shown every 5 values with no labels.
colors SliderTrackColors Specifies the colors that should be used on different parts of the track. This should be an object that can contain the following properties:

  • lower - Specifies the color or colors to be shown in the area before the first drag handle. This can be a string containing a color value or an array of strings which will result in a gradient.
  • range - Specifies the color or colors to be shown in the area between the first and second drag handle (the selected range of values). This only applies when a range of values can be selected. This can be a string containing a color value or an array of strings which will result in a gradient.
  • higher - Specifies the color or colors to be shown in the area after the last drag handle. This can be a string containing a color value or an array of strings which will result in a gradient.

SliderTicksOptions

Property Type Description
snap SliderSnap Specifies if the slider should snap to the ticks. There are several available options:

  • SliderSnap.None - No snapping will occur. This is the default value.
  • SliderSnap.Major - Will snap to the major ticks.
  • SliderSnap.Minor - Will snap to the minor ticks.
  • SliderSnap.All - Will snap to both the major and minor ticks.
major SliderTickOptions Should be an object containing properties that configure the major ticks. Major ticks will be more prominent than minor ticks and tend to be less frequent.
minor SliderTickOptions Should be an object containing properties that configure the minor ticks. Minor ticks will be less prominent than major ticks and tend to be more frequent.

SliderTickOptions

Property Type Description
show boolean Specifies whether this variety of ticks should be shown. The available options are true or false. By default both major and minor ticks will be shown.
steps number | number[] Specifies at what interval this variety of tick should appear. Default for major ticks is 10, the default for minor ticks is 5. Additionally this can be an array containing precise values of when ticks should appear. This can be useful when ticks should not appear at regular intervals.
labels boolean Specifies whether or not this variety of ticks should display labels. The available options are true or false. By default major ticks will show labels, minor ticks will not.
formatter (value: number) => string | number This allows you to specify a function that will format the value in the tick label. This should accept a value as a parameter and return a modified value. By default the value will not be modified.
