The following attributes can be set to customize the behavior of the control.

In an Angular application, ngModel may be used to bind the control to a form.

Name Type Description
@Input @Output value any Determines if the checkbox should be checked, unchecked or indeterminate. The default value is false.
@Input indeterminateValue any If value is set to the indeterminate value specified using this attribute, it will neither display the checkbox as checked or unchecked, and will instead show the indeterminate variation. The default value is -1.
@Input simplified boolean If set to true the checkbox will be displayed without a border and background. The default value is false.
@Input clickable boolean If set to true the checkbox will not toggle state when clicked. The default value is false.
