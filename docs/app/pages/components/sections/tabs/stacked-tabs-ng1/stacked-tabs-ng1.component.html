The stacked tabs can be present on left side or right side of the content. The left-stacked tabs can be created by enclosing the HTML of normal tabs with a div .tabs-left. The right-stacked tabs can be created by enclosing the HTML of normal tabs with a div .tabs-right. The align attribute of the tab element can be used to set the alignment of text and icon in the tab heading. This attribute can have value as text-center, text-left or text-right. Other attributes of stacked tabs are same as the normal tabs.
