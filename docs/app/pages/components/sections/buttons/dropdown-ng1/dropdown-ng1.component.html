On the div.btn-group set the dropdown attribute. The dropdown can either be opened by using the is-open attribute, or assigning the dropdown-toggle attribute to the desired .btn.

Set on-toggle="yourFunction(open)" for yourFunction to be executed every time the dropdown changes state.

It is possible to add interactive items to the dropdown. If the dropdown should remain open when clicking on an item, add ng-click="$event.stopPropagation()" to the anchor element. The dropdown-indent class is provided to visually distinguish subordinate controls within the dropdown, as seen in the first example.

Menu Features

Styles are provided to create menus with common features including gutter icons, keyboard shortcut hints, and submenus.

To style menu items as above, apply the dropdown-menu-item class to the a element of the menu item. This will contain up to three span elements with classes as follows:

  • dropdown-menu-icon - This contains the icon, which may be one of the UX Aspects Icons or an image. If no icon is required, leave this empty to ensure a consistent gutter for other items with icons.
  • dropdown-menu-text - This contains the menu text.
  • dropdown-menu-right - Optionally, this may contain right aligned text or icons. dropdown-menu-hint provides a default text style for keyboard shortcut hints here.

See Keyboard Service for information on binding keyboard shortcuts.

Set the width on the ul element to ensure that the menu contents will fit.

To create a menu item which opens a submenu, set the dropdown-submenu class on the li element, and insert the menu contents below the a element. The above example demonstrates both a standard and a custom submenu; see the example code for details.
