To create checkbox buttons create a .btn-group with the required .btns & including button-toggle-primary or button-toggle-accent class inside.

On each .btn set ng-model to the property required and specify the btn-checkbox attribute. Use the btn-checkbox-true and btn-checkbox-false attributes to specify true and false values on a per button basis.

For assistive technologies you should use the role="checkbox" attribute, together with the aria-checked attribute. The aria-checked attribute should be bound to the same property you use for the checked state.

Additionally you can include a span with the sr-only class to convey additional information on the state of the buttons. This information will be read by screen-readers but not visible on the page.
