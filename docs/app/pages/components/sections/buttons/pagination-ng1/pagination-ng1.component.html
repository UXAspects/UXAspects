Create a pagination element and set the total-items to the total number of items and set ng-model to the current page being displayed.

To trigger a function call when the page is changed set ng-change to the function e.g. ng-change="pageChanged()".

We recommend accompanying the pagination control with a label like “Page X of Y”.

Name Type Binding Description Optional
total-items number variable The total number of items in all pages. false
items-per-page number variable Maximum number of items per page. A value less than one indicates all items on one page. Defaults to 10 if not set. true
current-page number variable The currently selected page. false
num-pages number variable The total number of pages, this is a read only property which can be used for displaying the total number of pages. true

Note: Refer to UI Bootstrap pagination for more options.
