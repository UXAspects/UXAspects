The EBox panel is used throughout our documentation to contain our examples and code snippets. The surrounding panel of this example is an EBox panel. To create an EBox you can use the ux-ebox component.

Within this component you should have two child components, the ux-ebox-header component and the ux-ebox-content component. Inside these you can insert HTML to be displayed in that region of the EBox component.
