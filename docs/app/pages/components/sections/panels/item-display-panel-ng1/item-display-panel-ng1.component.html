Each item that opens the item display panel is contained in the display-panel-item directive.

The shadow property is optional and can be added to the display-panel-item to display a curved shadow under the header.

The auto-index attribute is used to enable keyboard controls - full documentation can be found on the Table Indices page.

The input value of the directive is an object. It is defined by the panel property of each item object in vm.items which in turn defines the item display panel.

The following defining the modal are set in the panel property of each item.

Property Description
title Defines the Header title of the item display panel.
main Provides a path to a template representing item display panel content.
footer Provides a path to a template representing item display panel footer content.
modalColumns Classes defining the number of columns as per the grid system for a responsive item display panel or a class with a preset width.
top Number, defines the initial top position of the modal with respect to standard header, condensed header or toolbar.
reference Element id or class, item display panel is displayed in reference to this element after scroll.
scope A scope instance to be used for the modal's main content.
animate (Optional) A boolean value to set the panel to animate in and out of the screen.

The next and previous buttons can be added to the panel footer to navigate between the items. The ng-click attribute of the previous button should be set as the previous() function and the ng-click attribute of the next button should be set as the next() function. The previous and next buttons will be disabled accordingly if the ng-disabled attribute is set to previousBtnStatus and nextBtnStatus respectively.
