To add the item display panel component to your application you need to import the ItemDisplayPanelModule which allows you to use the ux-item-display-panel component.

The following attributes can be used to customise the behaviour of the component.

Name Type Description
@Input @Output visible boolean Defines whether the item display panel is visible, if using two-way binding this will update when the item display panel is shown and hidden.
@Input title string? The title displayed at the top of the item display panel.
@Input top number? The number of pixels from the top of the screen, this can be used to keep the item display panel below the page header. Default: 0.
@Input animate boolean? Defines whether the item display panel will animate in and out from the side of the screen. Default: false.
@Input shadow boolean? Defines whether the item display panel will have a rounded shadow visible under the header. Default: false.

To add the main body of content to the item display panel the uxItemDisplayPanelContent directive can be added allowing you to add any required content below the header.

To add a footer to the item display panel the uxItemDisplayPanelFooter directive can be added allowing you to add any required content to the bottom of the item display panel
