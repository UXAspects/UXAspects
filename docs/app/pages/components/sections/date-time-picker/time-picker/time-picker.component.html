The timepicker can be created using the timepicker element. The ngModel should be set to the Date object that provides the time state. The min and max attributes define minimum or maximum time selectable by the timepicker. The hourStep and minuteStep attributes indicate the number of hours or minutes to increase or decrease when using a button respectively. The showMeridian attribute can be used to display a 12-hour clock. An additional component which displays seconds can be included using the showSeconds attribute. readonlyInput can be used to control the disabled state of the time picker. mousewheel and arrowkeys will determine whether using mousewheel or arrow keys respectively to change the time is allowed. A callback function can be provided to the ngModelChange attribute which will be called when the model changes.
