The navigation menu service ($navigationMenu) can be used in combination with the navigation menu button to control the visibility of the side navigation menu from anywhere in the application.

The following code shows an example of using this service; the full example can be seen via the CodePen link above.

The following functions are available on the $navigationMenu service.

Name Description
show Shows the side navigation menu.
hide Hides the side navigation menu.
visible Returns true if the side navigation menu is currently visible; otherwise false.
collapseAtWidth Returns the minimum window width (in pixels) at which the side navigation menu is displayed. At window widths narrower than this value, the menu will be hidden.
setCollapseAtWidth(width) Set the minimum window width (in pixels) at which the side navigation menu is displayed. Set the value to 0 to prevent hiding the side navigation menu based on window width.
setDefaultCollapseAtWidth Set the collapseAtWidth value to its default.

The event navigationMenu.visibilityChanged is also broadcast on the root scope whenever the visible state changes. The data parameter of this event is a boolean indicating whether the side navigation menu has become visible.
