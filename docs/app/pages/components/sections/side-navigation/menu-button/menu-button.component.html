The menu button on the top left of the navigation menu is used to toggle the left navigation menu. Add the minimize-menu directive to an element to initiate toggle.

Menu button is contained in the nav-menu-btn-container and nav-menu-toggle-btn classes.

The menu button can be styled by adding nav-menu-btn-container-dark or nav-menu-btn-container-light class to the root element for dark and light themes respectively.

When the menu is hidden, add the nav-menu-btn-container-top-light or nav-menu-btn-container-top-dark class to the root element to indicate light or dark theme with respect to the top header theme.

Add the option animate to make the navigation bar slide in and out.

The navigation menu may be collapsed by default using the auto-collapse attribute which will only show the menu when toggled by the user and then collapse again as soon as a page has been selected.
