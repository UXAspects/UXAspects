The following attributes can be used to customize the popovers appearance and behavior:

Property Type Binding Description Optional
popover-template string variable This defines the template file to be shown in the popover. false
popover-placement string literal This defines the position of the popover. Available options are top, bottom, left and right true
popover-class string literal This defines any additional classes to be added to the popover to allow custom styling. true
popover-fixed boolean literal This defines whether the popover should have its position set to fixed. true
popover-offset-x | popover-offset-y number literal This defines the number of pixels to move the popover from its default position. true
popover-trigger string literal This defines what events will open and show the popover. Default is click (See popover section for all options). true
popover-append-to-body boolean literal This defines whether or not the popover should be added to the body element or as a sibling to the element with the popover attributes. true
