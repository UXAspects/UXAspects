The following properties can be set on columns to customize their appearance:

Property Type Description Optional
title string This defines the title to be displayed in the column header. false
template string This defines the HTML to be used in the cells in this column. This property is required if the templateUrl property is not defined. true
templateUrl string This defines a path to an HTML file to be used in the cells in this column. This property is required if the template property is not defined. true
width number This defines the width of the column. If not specified the column will be automatically sized. true
