A progress bar can be created by using the ux-progress-bar component. Custom content can be added to the bar by adding HTML inside the ux-progress-bar tag. The following attributes can be used to configure the appearance and behavior of the progress bar:

Name Type Description
@Input value number The value the bar should represent. This will be compared to the max value to determine the width of the bar.
@Input max number The upper limit of the bar. The default value is 100.
@Input trackColor string The color of the track. The default value is grey7.
@Input barColor string The color of the bar. The default value is accent.
