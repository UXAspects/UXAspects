The Sankey Chart can be used to visualize the flow of data or items to various nodes. An ng-template with the identifier of #sankeyNodeTemplate should be added as a child element to define the content of the nodes.

The following options can be used to configure the chart:

Define the nodes to display in the chart. Define the nodes to display in the chart. Define the title for each column. Define the minimum width of a node. Define the maximum width of a node. Define the content of the tooltip that appears when hovering over a link. Define the content of the tooltip that appears when hovering over a falloff indicator. Define a unique ID for this node. Define additional custom data for this node. This can be accessed in the template and used to display additional information about a node to the user. Define the ID of the source node. Define the ID of the target node. Define the value that this link represents. The data that represents the node. The current active state of the node. The current focus state of the node.
