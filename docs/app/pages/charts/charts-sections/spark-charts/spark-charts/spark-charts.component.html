A spark chart can be created by using the ux-spark directive. The spark chart appearance and behavior can be customized by using the following attributes:

Defines the color of the bar and the track. This should specify a color from our color palette. Defaults to the chart1 color in the color palette and with an opacity of 0.2 for the track. Defines the color of the bar. Defaults to the chart1 color in the color palette. Enhances customization by permitting arbitrary colors to be specified. For multi-value spark charts multiple colors can be specified in an array and should match the number of values supplied. Defines the color of the track. Defaults to the chart1 color in the color palette with an opacity of 0.2. Enhances customization by permitting arbitrary colors to be specified. Indicates the current value of the spark chart. This should be a value between 0-100. Default value is 0. For multi-value spark charts multiple values can be specified in an array and should match the number of bar colors supplied. Indicates the height of the bar in pixels. Defines the text to be displayed inline with the bar. Defines the text to be displayed in the top left region of the chart. Defines the text to be displayed in the top right region of the chart. Defines the text to be displayed in the bottom left region of the chart. Defines the text to be displayed in the bottom right region of the chart. If defined, this text will be displayed in a tooltip when the mouse is over the bar.
