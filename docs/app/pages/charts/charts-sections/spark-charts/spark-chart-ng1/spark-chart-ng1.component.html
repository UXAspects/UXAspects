Spark Line indicator can be created using the spark directive. The attributes of the spark line are explained below:

Defines the class name that is used to specify the spark line color. Defines the color of the bar. For multi-value spark charts multiple colors can be specified in an array and should match the number of values supplied. Indicates the current value of the spark line (progress). For multi-value spark charts multiple values can be specified in an array and should match the number of barColors supplied. Indicates the current value of the spark fill (height). Defines the spark line label and works if inline. Indicates the top margin alignment of the spark line. Defines the top left label value. Defines the top right label value. Defines the bottom left label value. Defines the bottom right label value. Defines the value of the tooltip on the spark chart.

UX Aspects offers a wide range of chart colors. Set the color of the chart by setting the type property to one of the following:
