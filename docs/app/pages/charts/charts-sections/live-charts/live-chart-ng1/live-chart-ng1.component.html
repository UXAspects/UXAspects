Property Type Description Optional
data number or array Coordinates in the Live chart. false
series string Defines the chart type and its layout. Values include 'lines', 'bar' and 'points'. false

The following attributes should be defined within series

Property Type Description Optional
lineWidth number Defines the thickness of the line chart. true
show boolean If set to true displays the chart. false
fillColor array Specifies the colors of the fill. For Line chart this produces area graphs. If more than one color is specified this will produce a gradient. true
fill boolean or number If set to true fills the chart with colors in the fillColors property. A number between 0 and 1 defines the opacity of the fill. true
shadowSize number Default size of shadows in pixels. Set it to 0 to remove shadows. true

Live Charts are defined by the flot directive. The dataset and options attribute define the live chart data and options respectively. Dataset and Options are defined within the LiveChartCtrl.
