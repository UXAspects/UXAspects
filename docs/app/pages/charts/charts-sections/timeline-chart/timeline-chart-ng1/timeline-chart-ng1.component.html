Timeline Charts are defined by the flot directive. The dataset and options attributes define the chart data and options respectively. Dataset and Options are defined within the TimelineChartCtrl.

The lineDataService can be used to add zero-points to the data. This makes the y coordinates as zero when there is no data within a specified offset and the graph will lie at the x-axis within this interval.

The lineDataService contains the method addZeroPoints which should be called to add the zero points to the data. This method accepts two parameters:

Property Type Description Optional
datapoints number or array Returns data points with the zero points added. The returned data is an array of coordinates. false
offset number The zero points are added if there is an interval between the coordinates greater than the offset value passed. The default offset value is 1. true

An example of how the lineDataService can be used in the controller is shown below:

Property Type Description Optional
data number or array It is an array of coordinates in Timeline Chart. E.g. [ [x1, y1], [x2, y2], ... ] false
lines object Defines the parameters of the axis line. true

The following attributes should be defined within lines.

Property Type Description Optional
show boolean If set to true displays the chart. false
fill boolean or number If set to true fills the chart with colors in the fillColors property. A number between 0 and 1 defines the opacity of the fill. true
fillColor array(colors) Specifies the colors of the fill. If more than one color is specified this will produce a gradient. true
lineWidth number Defines the thickness of the line or outline in pixels. true
shadowSize number Default size of shadows in pixels. Set it to 0 to remove shadows. true
Property Type Description Optional
xaxes array Defines if more than one x-axis is used. true
yaxes array Defines if more than one y-axis is used. true
timeline array Helps configure the selection area on the chart. true

The following attributes should be defined within xaxes.

Property Type Description Optional
mode null or time Defines how data is represented on the x-axes. true
min number Defines the minimum value of the x-axis. true
max number Defines the maximum value of the x-axis. true
tickColor null or string(color) Specifies the colors of the ticks in the x-axis. true

The following attributes should be defined within yaxes.

Property Type Description Optional
show boolean If set to true displays the axis. true
min number Defines the minimum value of the y-axis. true
max number Defines the maximum value of the y-axis. true

The following attributes should be defined within timeline.

Property Type Description Optional
color string Specifies the color of the selected area. true
start number Defines the position on the x axis that the selection should initially start. true
end number Defines the position on the x axis that the selection should initially end. true
keyboardNavigation boolean Defines if arrow keys are allowed to move selected area. true
enabled boolean If set to true this will enable selection on the grid. true
zoom array Helps configure chart zooming. true
dragHandles array This helps configure the handles on either end of the selection that the user can drag. true
callback functional This will be called whenever the selected range changes. It should accept one argument that will be an object containing two values. true

The following attributes should be defined within zoom.

Property Type Description Optional
enabled boolean Defines if chart should zoom when the range is changed. true
minimumRange number Defines the smallest range the chart will allow to be selected. true

The following attributes should be defined within dragHandles.

Property Type Description Optional
width number Defines the width of the handles. false
color string Defines the color of the handles. false
tooltips array Helps configure tooltips on the selection region. false
tooltipFormatter function Use this function to define the tooltip text. It should accept one parameter which will be the value on the x axis and should return a string. true

The following attributes should be defined within tooltips.

Property Type Description Optional
enabled boolean If set to true shows the tooltip on the selection region based on the onHover, onDrag and onDrag options. false
onHover boolean If set to true the tooltip will show when the cursor is over the selection region. true
onDrag boolean If set to true the tooltip will show when the selection region is being dragged/resized. true
onDragEnd boolean If set to true the tooltip will show when the chart has finished being dragged/resized but the mouse is still pressed. true

The following attributes should be defined within callback.

Property Type Description Optional
start number The position on the x-axis where the selected area starts. false
end number The position on the x-axis where the selected area ends. false
