Timeline Charts are defined by the flot directive. The dataset and options attributes define the chart data and options respectively. Dataset and Options are defined within the TimelineChartCtrl.

The lineDataService can be used to add zero-points to the data. This makes the y coordinates as zero when there is no data within a specified offset and the graph will lie at the x-axis within this interval.

The lineDataService contains the method addZeroPoints which should be called to add the zero points to the data. This method accepts two parameters:

Returns data points with the zero points added. The returned data is an array of coordinates. The zero points are added if there is an interval between the coordinates greater than the offset value passed.

Specifies the color of the selected area. Defines the position on the x axis that the selection should initially start. Defines the position on the x axis that the selection should initially end. This helps configure the handles on either end of the selection that the user can drag. This will be called whenever the selected range changes. Defines if chart should zoom when the range is changed. Defines the smallest range the chart will allow to be selected.

Defines the width of the handles. Defines the color of the handles. Helps configure tooltips on the selection region. Use this function to define the tooltip text. It should accept one parameter which will be the value on the x axis and should return a string.

The following attributes should be defined within tooltips.

If set to true shows the tooltip on the selection region based on the onHover, onDrag and onDrag options. If set to true the tooltip will show when the cursor is over the selection region.
