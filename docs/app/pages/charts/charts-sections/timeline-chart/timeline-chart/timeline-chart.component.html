Timeline Charts can be added using the ng2-charts library. The chart.js library needs to be imported and the ChartsModule needs to be added to the appropriate NgModule.

To add Timeline functionality the TimelineChartPlugin should be imported from the @ux-aspects/ux-aspects library and the TimelineChartPlugin.register() function should be called.

The a timeline object should be added to the chart options to configure the chart:

Specify a background color for the chart area. Specify a color for the selected region. Specify the selected range and range limits. Specify a callback that will be triggered when the selected range changes. Specify options related to keyboard accessibility. Specify the appearance of the drag handles. Specify the lower boundary of the selection. Specify the upper boundary of the selection. Specify the minimum allowed range in milliseconds. Specify the maximum allowed range in milliseconds. Specify the handle background color. Specify the handle foreground color. Specify the handle focus indicator color. Specify how much a handle should move (in milliseconds) when an arrow key is pressed.
