Property Type Description Optional
label string Label name specified in the legend. true
data number or array An array of coordinates in the chart e.g. [ [x1, y1], [x2, y2], ... ] false
forecastData number or array An array of coordinates defining the forecast or future data. true

Line Charts are defined by the flot directive. The dataset and options attribute define the line chart data and options respectively. Dataset and Options are defined within the LineChartCtrl.

Note: Refer to the Flot Options page for details of chart options.

To convey chart information to users with assistive technologies, you can include a text version of the data. Use the sr-only class to make the text hidden but visible to screen readers, and give the container a tabindex attribute so it can be reached with the keyboard, and place it near the chart.

Note: To upgrade this component to use in an Angular application use: upgradeAdapter.upgradeNg1Component('uxFlotNg1')
