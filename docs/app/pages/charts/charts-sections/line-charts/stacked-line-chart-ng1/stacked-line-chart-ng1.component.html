Stacked Line Charts are defined by the flot directive. The dataset and options attributes define the chart data and options respectively. Dataset and Options are defined within the StackedLineChartCtrl.

If set to true shows the chart in stacked form.

Note: Refer to the Flot Options page for details of chart options.

Stacked line charts are an enhanced version of line charts. For more information on options, refer to the line chart documentation.

To convey chart information to users with assistive technologies, you can include a text version of the data. Use the sr-only class to make the text hidden but visible to screen readers, and give the container a tabindex attribute so it can be reached with the keyboard, and place it near the chart.
