Label name specified in the legend. A number in the donut chart defining the area of each donut section. Defines the color of each donut section. Default color of the translucent overlay used to highlight on mouse hover.

Donut Charts are defined by the flot directive. The dataset and options attribute define the donut chart data and options respectively. Dataset and Options are defined within the DonutChartCtrl.

Defines the chart type and its layout. Values include 'lines', 'bar' and 'points', e.g. if data represents a donut chart, then the value is set to 'pie'. Helps configure the grid options.

Grid options are explained below.

To convey chart information to users with assistive technologies, you can include a text version of the data. Use the sr-only class to make the text hidden but visible to screen readers, and give the container a tabindex attribute so it can be reached with the keyboard, and place it near the chart.
