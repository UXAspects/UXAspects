The available options are:

Defines the diameter of the chart. This will also accept a percentage value in the form of a string, e.g. "50%". Defines the width of each ring. Defines the spacing between each ring. Defines whether or not ring should show a hover effect when the mouse is over it. Defines a function to be called when a ring is hovered. The function should accept one argument that will contain the data corresponding to the hovered ring. Defines a function to be called when a ring is clicked. The function should accept one argument that will contain the data corresponding to the clicked ring. Defines the content and appearance of a center label if one is defined. Defines tooltip options.

The following attributes should be defined within centerLabel.

Whether or not the chart should show a center label. Defines the content of the center label. Defines the color of the center label text. Defines the font family of the center label. Defines the font size of the center label. Take into account the size of the chart when considering a font size to ensure text is suitably displayed within the chart.

The following attributes should be defined within tooltip.

Defines whether a tooltip should show when a ring is hovered. Defines the content of the tooltip. The function should accept one argument which will contain the data corresponding to the ring that was hovered. The function should return a HTML string. Defines how the tooltip should be offset from the mouse position.

The following attributes should be defined within shifts.

Defines how many pixels left or right the tooltip should be positioned from the mouse position. Defines how many pixels up or down the tooltip should be positioned from the mouse.

Nested Donut Charts are defined by the nested-donut directive. The dataset and options attributes define the nested donut chart data and options respectively. Dataset and Options are defined within the NestedDonutChartCtrl.

Note: Refer to the Flot Options page for details of chart options.

Note: This component can be used in an Angular application by importing the HybridModule.
