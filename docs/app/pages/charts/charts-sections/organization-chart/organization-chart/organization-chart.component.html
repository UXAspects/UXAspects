The organization chart can be used to visually present an explorable hierarchical dataset.

The content of the nodes should be defined using an ng-template identified as #nodeTemplate that is a child of the component element.

The following options are available to customize component:

Define the dataset to display in the chart. Define the presentation of the lines connecting nodes. Define the width of nodes in pixels. Define the height of nodes in pixels. Define the duration of animations in milliseconds. Define if the reveal more button should be visible. This is used to display ancestor nodes, which may be initially hidden. Define the aria label of the reveal more button. Define the selected node. By default the root node will be selected. Will emit when a node is selected. Will emit when the reveal button is clicked. Use this handler to modify the dataset with additional nodes. Expand a node. Collapse a node. Toggle the expanded state of a node. A unique identifier for this item. Provide custom data for this item. This can be used to display information within the node template. Define the children of this node if there are any. Define the expanded state of this node. The node object that this item represents. The custom data that was provided for this node. The focused state of the node.
