Bar charts and line charts can be extended to have scrolling capabilities for cases when the chart has to display large amounts of data.

To make a chart scrollable we need to define a scrollable object under the xaxis options.

Property Type Description Optional
scrollable array Helps configure the scrolling behavior. true

The following attributes should be defined within scrollable

Property Type Description Optional
enabled boolean Set to true if scrolling will be enabled. true
range number Represents the size of each page in relation to the points on the x-axis. For example, in the chart above a value of 4 will initially show any points between 0.5 and 4.5. When the next button is clicked it will show points between 4.5 and 8.5 true
lowerBound number Represents the minimum point on the x-axis that can be scrolled to. Once the chart is at this position the scroll left button will be hidden. true
upperBound number Represents the maximum point on the x-axis that can be scrolled to. Once the chart is at this point the scroll right button will be hidden. true
color color Sets the color of the scroll buttons. true
callback function Will be called when the chart is scrolled. It should take in one parameter that will contain two properties. true

The following attributes should be defined within callback

Property Type Description Optional
start number Contains the minimum value on the x-axis that is currently visible. true
end number Contains the maximum value on the x-axis that is currently visible. true

Note: Refer to the Flot Options page for details of chart options.

Scrollable charts are an enhanced version of bar/line charts. For more information on adding a bar or line chart, refer to the appropriate chart documentation.
