The available options are:

Property Type Description Optional
size number Defines the diameter of the chart. (default: 150). This will also accept a percentage value in the form of a string, e.g. "50%". true
donutWidth number Defines the width of each ring. (default: 4) true
donutSpacing number Defines the spacing between each ring. (default: 4) true
hoverAnimation boolean Defines whether or not ring should show a hover effect when the mouse is over it (default: true) true
onHover function Defines a function to be called when a ring is hovered. The function should accept one argument that will contain the data corresponding to the hovered ring. true
onClick function Defines a function to be called when a ring is clicked. The function should accept one argument that will contain the data corresponding to the clicked ring. true
centerLabel object Defines the content and appearance of a center label if one is defined. true
tooltip object Defines tooltip options. true

The following attributes should be defined within centerLabel.

Property Type Description Optional
show boolean Whether or not the chart should show a center label. true
text string Defines the content of the center label. true
color color Defines the color of the center label text. true
font string Defines the font family of the center label. true
fontSize number Defines the font size of the center label. Take into account the size of the chart when considering a font size to ensure text is suitably displayed within the chart. true

The following attributes should be defined within tooltip.

Property Type Description Optional
show boolean Defines whether a tooltip should show when a ring is hovered (default: true) true
content function Defines the content of the tooltip. The function should accept one argument which will contain the data corresponding to the ring that was hovered. The function should return a HTML string. false
shifts object Defines how the tooltip should be offset from the mouse position. true

The following attributes should be defined within shifts.

Property Type Description Optional
x number Defines how many pixels left or right the tooltip should be positioned from the mouse position (default: 0) true
y number Defines how many pixels up or down the tooltip should be positioned from the mouse position (default: 0) true

Nested Donut Charts are defined by the nested-donut directive. The dataset and options attributes define the nested donut chart data and options respectively. Dataset and Options are defined within the NestedDonutChartCtrl.

Note: Refer to the Flot Options page for details of chart options.
