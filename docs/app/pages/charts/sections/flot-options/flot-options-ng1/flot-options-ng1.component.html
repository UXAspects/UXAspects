Listed below are the options which can be specified in the controller for a chart:

Property Type Description Optional
series string Defines the chart type and its layout. Values include 'lines', 'bars', 'points', 'pie' etc. E.g. If data represents a bar chart, then the value is set to 'bars' etc. false
xaxis array Configures the X-axis with options like show, color, etc. true
yaxis array Configures the Y-axis with options. true
margin object Defined with "top", "left", "bottom" and "right", space in pixels between the canvas edge and the grid. true
markings array Markings, used to draw simple lines and rectangular areas in the background of the plot. It can be an array of ranges in the form of { xaxis: { from, to }, yaxis: { from, to } } or a function(above example) that returns this range (note: markings are only used for Live Charts).
colors array(colors) Specifies colors used to represent a chart. true
grid array Helps configure the grid options true
legend array Defines the data series labels and label boxes with the color of the series true
tooltip array Defines how a tooltip should be shown true
onPlotClick function Called when plot area is clicked. Parameters are event, the position and a nearby data item object. This option works when the "clickable" attribute is set to true true
onPlotHover function Called when plot area is hovered. Parameters are event, the position and a nearby data item object. This option works when the "hoverable" attribute is set to true. If "autoHighlight" is true (the default), nearby data items are highlighted automatically. true

The following attributes should be defined within series

Property Type Description Optional
show boolean If set to true the chart is displayed. false
fillColor array(colors) Specifies the colors of the fill. If more than one color is specified this will produce a gradient. true
fill boolean or number If set to true fills the chart with colors in the fillColors property. A number between 0 and 1 defines the opacity of the fill. true
lineWidth number Defines the thickness of the line or outline in pixels. true
align string Centers the axis labels if set to `center`. true
highlightColor color Default color of the translucent overlay used to highlight on mouse hover. true
forecastFillColor color Defines the fill color for the portion representing forecast data. true
forecastColor color Defines the stroke color for the portion representing forecast data. true
forecastHighLightColor color Default color of the translucent overlay used to highlight on mouse hover for the portion representing forecast data. true
forecastDashStyle array Default is [5], defines a list of numbers that specifies distances to alternately draw a line and a gap (in coordinate space units). Similar to this property in HTML5 Canvas. true
shadowSize number Default size of shadows in pixels. Set it to 0 to remove shadows. true
horizontal boolean If true the bars will be presented horizontally (note: this can only be set for Bar Charts). true
stack boolean If set to true shows the chart in stacked form (note: this can only be used to create Stacked Bar/Line Charts). true
valueLabels array Allows labels to be displayed beside bars or data points. true

The following attributes should be defined within valueLabels

Property Type Description Optional
show boolean When set to true labels will be displayed. true
valign string Defines the vertical alignment of the text. The available options are: 'top', 'above', 'below', 'middle' or 'bottom'. true
plotAxis string Defines the axis you wish to display the values of. The available options are: 'x' or 'y'. true
xoffset number Defines the number of pixels the label should move to the left or right of its original position. true
yoffset number Defines the number of pixels the label should move up or down from its original position. true
font string Defines the font size and font family the labels should use. true
fontColor color Sets the text color of the labels. true
labelFormatter function Should accept one argument that will contain the axis position of the associated data. The function can be used to change the text in the label and should do this by returning the desired text. true

The following attributes should be defined within xaxis

Property Type Description Optional
show boolean If set to true displays the axis. true
tickDecimals null or number Default is auto-detected, describes ticks of the axis. true
min number Set the start value of the x-axis. true
max number Set the end value of the x-axis. true
color color Defines the color of the axis lines. true
ticks array Sets the x-axis labels to a specified value. true
tickFormatter function Defines how ticks are formatted. true
mode null or time Defines how data is represented on the x-axis. true
position string Set as "top" or "bottom", specifies the position of the axis. true

The following attributes should be defined within yaxis

Property Type Description Optional
show boolean If set to true displays the axis. true
min number Set the start value of the y-axis. true
max number Set the end value of the y-axis. true
tickFormatter function Defines how ticks are formatted. true
alignTicksWithAxis null or number Aligns the ticks of this axis with the ticks of the other axis. true
position string Set as "right" or "left", specifies the position of the axis. true

The following attributes should be defined within grid

Property Type Description Optional
color color Specifies the color of the grid lines. true
hoverable boolean If set to true enables interactivity in the grid. true
clickable boolean If set to true enables interactivity in the grid. true
borderWidth number or object Defines the width of the grid border. true
tickColor null or color Specifies the colors of the ticks in an axis. true
borderColor color Defines the border grid lines. true
preventOverlap boolean If set to true this will prevent any x-axis labels from overlapping. true
minBorderMargin number or null Specifies the default minimum margin around the border. Used to make sure that points aren't accidentally clipped by the canvas edge. true
backgroundColor color or object Specifies the background color or background gradient colors of the chart. true

The following attributes should be defined within legend

Property Type Description Optional
show boolean If set to true shows the legend. true
container string The id or class name of the container to hold the legend. true
noColumns number The number of columns for the legend. true
position string "ne" or "nw" or "se" or "sw", defines the position of the legend. true

The following attributes should be defined within tooltip

Property Type Description Optional
show boolean Set to true to show the tooltip. true
cssClass string Set to the name of a custom css class to style the tooltip. true
defaultTheme boolean Set to false if you are using a custom css class. true
snap boolean Set to true to have tooltip in fixed position. When set to false, the tooltip moves along with the mouse pointer. true
shifts array Values to shift tooltip position relative to the mouse pointer for x and y, negative values are allowed. true
content string or function Specifies the tooltip text e.g. "x : %x, y : %y" - displays x : x-value, y : y-value. Function can be used if you require more control over how the content is generated. Note that if you wish to access the axis labels rather than the number value (co-ordinates) assigned to them when using a function, you must use the "%" identifier followed by the axis (x or y) rather than the "xval" or "yval" parameters that are passed to the function. true
xDateFormat String provides the format of the x-axis Date. true
