<updating-linechart data="pc.UpdatingLineChart.data" method="pc.UpdatingLineChart.method" options="pc.UpdatingLineChart.options" updateinterval="pc.UpdatingLineChart.updateinterval"></updating-linechart>

In the template, pass pc.UpdatingLineChart.data to define the chart data, pc.UpdatingLineChart.options to define the chart rendering options, pc.UpdatingLineChart.method to define the method that generates updated values and pc.UpdatingLineChart.updateinterval to define the interval.

Updating Line Charts are rendered by passing the data, options, method and updateinterval parameters to the controller.

Name Type Description Optional
data array Array of data points for the chart to display. false
stroke string Provides the color of the stroke in a chart. true
fill string Written inside the options, these are the colors used to fill line or bar chart. true
width number Number, defines the width of the graph. true
height number Number, defines the height of the graph. true
method function Function that updates the line chart. Parameters: oldData; Return Parameter: newData. true
updateinterval number Milliseconds, time interval for re-generation. true

To provide accessibility for users with assistive technologies, include a span with the sr-only class which conveys the same information as the chart. For updating charts and line charts, the most appropriate information to provide will depend on the purpose of the chart.
