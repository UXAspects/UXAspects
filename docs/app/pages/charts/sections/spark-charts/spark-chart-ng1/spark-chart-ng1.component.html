Spark Line indicator can be created using the spark directive. The attributes of the spark line are explained below:

Property Type Description Optional
type string Defines the class name that is used to specify the spark line color. true
value number Indicates the current value of the spark line (progress). false
fillheight number Indicates the current value of the spark fill (height). false
inlineLabel HTML Defines the spark line label and works if inline. true
top number Indicates the top margin alignment of the spark line. true
topLeftLabel string Defines the top left label value. true
topRightLabel string Defines the top right label value. true
bottomLeftLabel string Defines the bottom left label value. true
bottomRightLabel string Defines the bottom right label value. true
spark-tooltip string Defines the value of the tooltip on the spark chart. true

You can also create your own class to use for the spark colors. Name your class and specify a background-color. Within this class you must specify a fill class which also has a background-color set as the fill for the spark chart. See an example below:

Multiline tooltips can be added to the Spark Line indicator using the `spark-tooltip` attribute to the spark directive.

Colors are defined from the set of chart colors and the pre-defined class choices are as follows:

Property Type Description Optional
spark-chart1 color @chart1 fill, 0.1 background opacity. true
spark-chart2 color @chart2 fill, 0.2 background opacity. true
spark-chart3 color @chart3 fill, 0.2 background opacity. true
spark-chart4 color @chart4 fill, 0.2 background opacity. true
spark-chart5 color @chart5 fill, 0.2 background opacity. true
spark-chart6 color @chart6 fill, 0.2 background opacity. true
spark-ok color @ok fill, 0.25 background opacity. true
spark-warning color @warning fill, 0.25 background opacity. true
spark-critical color @critical fill, 0.25 background opacity. true
