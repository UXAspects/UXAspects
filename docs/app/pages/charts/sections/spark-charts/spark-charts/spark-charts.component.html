A spark chart can be created by using the ux-spark directive. The spark chart appearance and behavior can be customized by using the following attributes:

Property Type Description
barColor string Defines the color of the bar. Defaults to the chart1 color in the color palette.
trackColor string Defines the color of the track. Defaults to the chart1 color in the color palette with an opacity of 0.2.
value number Indicates the current value of the spark chart. This should be a value between 0-100. Default value is 0.
barHeight number Indicates the height of the bar in pixels. Default value is 10.
inlineLabel string Defines the text to be displayed inline with the bar.
topLeftLabel string Defines the text to be displayed in the top left region of the chart.
topRightLabel string Defines the text to be displayed in the top right region of the chart.
bottomLeftLabel string Defines the text to be displayed in the bottom left region of the chart.
bottomRightLabel string Defines the text to be displayed in the bottom right region of the chart.
tooltip string If defined, this text will be displayed in a tooltip when the mouse is over the bar.
