The organization chart is designed to fill the size of the parent element.

The following attributes can be used to configure the chart:

Property Type Binding Description Optional
data object variable The data the chart should display. The object can contain any properties, however any child items should be listed in an array under a children property. false
options object variable Allows you to configure the appearance and behavior of the chart. All available options can be found in the table below. false

Chart Options

All of the following options are optional, with the exception of the node template, and the default value will be used when an option has not been specified.

Property Type Description
transition number Specifies the duration of any animations used in the chart in milliseconds (default: 750).
levels number Specifies number of node levels to show initially. Eg. 1 will show the root node and any of its immediate children (default: 1).
reveal function If specified an arrow will appear above the root node. When clicked this function will be called. This could be used for updating the data in the chart to show more parent levels in the hierarchy. To hide the button after it has been clicked the function should return false.
hierarchyBar object Specifies the appearance and behavior of the breadcrumb bar along the top of the chart.
nodes object Specifies the appearance and behavior of the nodes displayed in the chart. See table below for more info.
connector string Specifies the style of connector, curved or elbow. If unspecified the curved connector will be used.
search object Specifies whether the search functionality should be enabled and how it should be configured.

Note: Refer to the Hierarchy Bar Options for ‘hierarchyBar’ object attributes.

Node Options

Property Type Description
template string The url of a template file that will be inserted into each node.
toggle boolean If true, then clicking anywhere within a node will toggle the expanded/collapsed state of the node. Setting it to false will require you to manually expand and contract nodes (default: true).
size object Specifies the size of each node in pixels. It should have a width and height property (default: width - 210, height - 90).

Search Options

Property Type Description
enabled boolean If true the search icon will appear at the top of the chart allowing a user to search all chart nodes and quickly navigate to them. (default: false)
template string This should specify the url to a custom template file for the typeahead results. The default template will show one line of text from a name property on a node. See UI Bootstrap for more information on typeahead templates.
placeholder string Specifies placeholder text to show in the search input field. (default: '')
query function This function controls which items are suggested for any given search query. The function should expect two arguments, the first is the search query and the second is a node model. This function should return true if the node meets the search criteria and false if it does not. By default the nodes are filtered based on a case insensitive match on a name property.
selected function Specifies a callback function that will be called when a search item has been selected. The select node model will be passed as an argument to the function.
key string The primary identity property on a node model. This will be displayed in the search input field when an item is selected. (default: 'name')

Node Template

The node template will be compiled with a scope containing the data specific to that node. This can be used by the template to display the information within the node. Additionally several functions are also exposed on the scope.

  • getChildCount - this will return the number of children the node has.
  • getExpanded - will return true if the node is in the expanded state otherwise will return false.
  • expandNode - will make the node reveal its children.
  • collapseNode - will make the node hide its children.
