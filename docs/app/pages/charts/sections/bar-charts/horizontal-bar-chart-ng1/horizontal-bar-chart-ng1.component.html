Property Type Description Optional
horizontal boolean If true the bars will be presented horizontally. true

Horizontal Bar Charts are defined by the flot directive. The dataset and options attributes define the chart data and options respectively. Dataset and Options are defined within the HorizontalBarChartCtrl.

Note: Refer to the Flot Options page for details of chart options.

Horizontal bar charts are an enhanced version of bar charts. For more information on options, refer to the bar chart documentation.

Refer to flot-valuelabels for more options on value labels.
