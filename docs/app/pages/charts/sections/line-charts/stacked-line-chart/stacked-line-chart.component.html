Stacked Line Charts can be added using the ng2-charts library. The chart.js library needs to be imported and the ChartsModule needs to be added to the appropriate NgModule.


The baseChart directive should be added to a canvas element and the following attributes can be used to customize the chart behavior and appearance:

Name Type Description
@Input datasets object[] Contains the data and properties for each data set to be displayed in the chart.
@Input chartType string Indicates the type of the chart. Possible options are: line, bar, radar, pie, polarArea, doughnut
@Input options ChartOptions Configures the appearance and behavior of all aspects of the chart.
@Input colors object[] Configure the color values to be used within the chart.
@Input legend boolean Determines whether or not the legend should be displayed.
@Output chartClick Fires when click on a chart has occurred, returns information regarding active points and labels
@Output chartHover Fires when mousemove (hover) on a chart has occurred, returns information regarding active points and labels.
